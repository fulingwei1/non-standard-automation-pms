### 1. 路由未注册

**成本管理模块**:
- `/api/v1/costs/*` 路由未注册到主路由器
- 所有成本相关端点返回404

**工时管理模块**:
- 部分工时端点路径可能不匹配
- 需要检查实际路由注册

### 2. 端点缺失

**认证模块**:
- 修改密码端点 (`/api/v1/auth/change-password`) 未实现或路径错误

- 某些端点允许空更新
- 某些端点允许未来日期
- 分页大小限制可能未生效

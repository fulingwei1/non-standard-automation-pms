## 分层架构

### 1. 前端层

| 职责 | 技术 |
|------|------|
| UI 组件 | React 19 + shadcn/ui + Radix UI |
| 样式 | Tailwind CSS 4 |
| 路由 | React Router 7 |
| 全局状态 | Zustand |
| 服务端状态 | TanStack React Query |
| 表单 | React Hook Form + Zod |
| 图表 | Recharts + Ant Design Plots |

### 2. API 路由层 (`app/api/v1/endpoints/`)

- 71 个端点模块，通过 `_safe_include()` 安全注册
- 单个模块加载失败不影响其他模块启动
- 路由只负责：参数解析、权限检查、调用 Service、返回响应

### 3. 业务服务层 (`app/services/`)

- 93 个服务模块，按领域组织
- 包含 11 个状态机实现（审批流程适配器）
- AI/ML 服务：成本预测、胜率预测、工时预测、风险评估

### 4. 数据模型层 (`app/models/`)

- 18 个领域分组，65+ 核心模型
- SQLAlchemy 2.0 类型安全 ORM
- Pydantic 2.x 输入/输出校验（`app/schemas/`）

## 核心架构决策

### 多租户隔离

采用**单数据库多租户**架构，三层隔离：
1. **数据库约束** — 所有业务表包含 `tenant_id` 字段
2. **ORM 自动过滤** — `TenantContextMiddleware` 注入租户上下文
3. **业务逻辑检查** — Service 层二次校验

### 认证与授权

- **JWT 双 Token** — Access Token (24h) + Refresh Token (7d)
- **API Key** — 服务间调用
- **2FA** — 可选 TOTP 二次验证
- **RBAC** — 125 个 API 权限，471 个角色-权限映射

### AI 集成

- **成本预测引擎** — 3 种算法：历史平均、线性回归、趋势预测（scikit-learn）
- **外部 LLM** — Moonshot AI (KIMI) + 智谱 AI (GLM) 用于售前分析、方案生成
- **异常检测** — 工时异常、成本超支自动预警

### 安全

- CORS 白名单限制
- CSRF Token 保护
- 速率限制（slowapi）：全局 100/min，登录 5/min
- 安全响应头（X-Frame-Options, CSP 等）
- 审计日志中间件
- bcrypt 密码哈希

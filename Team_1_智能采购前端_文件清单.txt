十、部署说明
================================================================================

环境要求:
- Node.js >= 16.0.0
- npm >= 8.0.0

安装依赖:
  cd frontend
  npm install

开发运行:
  npm run dev

生产构建:
  npm run build

环境变量:
  VITE_API_BASE_URL=http://localhost:8000/api/v1/purchase

路由配置:
  /purchase/suggestions                - 采购建议列表
  /purchase/suggestions/:id            - 采购建议详情
  /purchase/suppliers/performance      - 供应商绩效管理
  /purchase/suppliers/ranking          - 供应商排名
  /purchase/orders/:orderId/tracking   - 订单跟踪
  /purchase/quotations/compare         - 报价比价

## 系统概述

项目进度管理系统是专为非标自动化设备企业设计的项目管理工具，支持:

- 📊 **项目管理**: 项目全生命周期管理
- 📋 **任务管理**: WBS分解、甘特图、关键路径
- ⏱️ **工时管理**: 工时填报、审核、统计
- 👥 **负荷管理**: 人员负荷分析、资源调配
- ⚠️ **预警管理**: 风险预警、智能提醒

## 快速入门

### 登录系统

1. 打开浏览器，访问系统地址

## 功能模块

### 1. 项目管理

#### 查看项目列表

1. 点击左侧菜单"项目管理"
2. 可切换"表格"或"卡片"视图
3. 使用筛选条件查找项目

#### 创建新项目

1. 点击"新建项目"按钮
2. 填写项目信息:
   - 项目编号（可自动生成）
   - 项目名称
   - 客户
   - 项目级别
   - 项目经理/技术负责人
   - 计划周期
3. 点击"创建"

#### 项目看板

进入项目详情页，可查看:

- **进度概览**: 总体进度、阶段进度
- **任务统计**: 各状态任务数量
- **工时统计**: 计划vs实际工时
- **团队成员**: 项目参与人员
- **时间线**: 项目更新记录

### 2. 任务管理

#### WBS任务分解

1. 进入项目 > 点击"WBS"标签
2. 查看树形任务结构
3. 点击"添加任务"创建新任务
4. 选择父任务可创建子任务

#### 甘特图视图

1. 进入项目 > 点击"甘特图"标签
2. 拖拽调整任务时间
3. 连线设置任务依赖
4. 红色高亮显示关键路径

#### 更新任务进度

**方式一：快速更新**
1. 在任务列表点击"更新进度"
2. 拖动进度条或输入百分比
3. 填写工作内容
4. 点击"确定"

**方式二：详情页更新**
1. 点击任务打开详情面板
2. 修改进度、状态等信息
3. 保存更改

### 3. 我的任务

#### 任务视图切换

- **列表视图**: 表格形式查看所有任务
- **看板视图**: 按状态分列展示
- **日历视图**: 按日期查看任务

#### 任务筛选

- 按状态筛选: 全部/进行中/待开始/已完成/逾期/阻塞
- 按项目筛选: 选择特定项目
- 按关键词搜索

### 4. 工时管理

#### 填报工时

1. 点击"工时管理"菜单
2. 选择日期和任务
3. 输入工时和工作内容
4. 保存为草稿或直接提交

#### 周工时表

1. 切换到"周视图"
2. 在表格中直接填写每天工时
3. 底部显示每日合计
4. 批量提交一周工时

#### 工时审核（管理者）

1. 进入"待审核"页面
2. 查看下属提交的工时
3. 点击"通过"或"驳回"
4. 驳回时需填写原因

### 5. 负荷管理

#### 查看团队负荷

1. 点击"负荷管理"菜单
2. 查看团队成员负荷率
3. 红色表示超负荷(>100%)
4. 绿色表示有余量(<80%)

#### 负荷热力图

- 横轴: 时间周期
- 纵轴: 团队成员
- 颜色深浅表示负荷高低

#### 查找可用资源

1. 点击"可用资源"卡片
2. 查看负荷较低的人员
3. 可按技能筛选

### 6. 预警中心

#### 预警类型

| 类型 | 图标 | 说明 |
|------|------|------|
| 任务逾期 | 🔴 | 任务已超过截止日期 |
| 进度滞后 | 🟠 | 实际进度落后计划 |
| 即将到期 | 🟡 | 任务将在3天内到期 |
| 负荷过高 | 🟣 | 人员负荷超过110% |

#### 处理预警

1. 点击预警查看详情
2. 选择处理方式:
   - **处理中**: 正在解决
   - **已解决**: 问题已修复
   - **忽略**: 不需要处理
3. 填写处理备注
4. 确认提交

### 7. 报表中心

#### 项目汇总报表

- 项目进度分布
- 健康状态统计
- 项目进度明细表

#### 工时统计报表

- 工时趋势图
- 项目工时分布
- 加班工时统计

#### 人员绩效报表

- 任务完成排名
- 工时效率分析
- 按时完成率

## 常见操作

### 如何批量更新任务进度？

1. 进入项目甘特图
2. 按住Ctrl选择多个任务
3. 点击"批量更新"
4. 设置统一的进度值
5. 确认更新

### 如何设置任务依赖？

1. 在甘特图视图
2. 将鼠标移到任务条右侧
3. 拖拽连线到目标任务
4. 或在任务详情中添加前置任务

### 如何导出项目数据？

1. 进入项目详情页
2. 点击右上角"导出"按钮
3. 选择导出格式(Excel/PDF)
4. 选择导出内容范围
5. 下载文件

### 如何接收企业微信通知？

1. 确保已绑定企业微信账号
2. 系统会自动推送:
   - 任务分配通知
   - 逾期预警
   - 审批提醒
3. 在设置中可调整通知偏好

## FAQ

### Q: 进度是如何自动计算的？

A: 系统采用加权计算方式:
- 每个任务有权重(weight)
- 项目进度 = Σ(任务进度 × 权重) / Σ权重
- 父任务进度由子任务自动汇总

### Q: 什么是关键路径？

A: 关键路径是项目中最长的任务链路，决定了项目的最短完成时间。关键路径上的任务延期会直接导致项目延期。

### Q: 健康状态如何判定？

A: 基于进度绩效指数(SPI):
- SPI ≥ 0.95: 绿色(正常)
- 0.8 ≤ SPI < 0.95: 黄色(关注)
- SPI < 0.8: 红色(预警)

SPI = 实际进度 / 计划进度

### Q: 工时填报有时间限制吗？

A: 
- 只能填报当天及之前的工时
- 超过7天的工时需管理员审批
- 月度锁定后不可修改

### Q: 如何处理任务阻塞？

A: 
1. 将任务状态改为"阻塞"
2. 填写阻塞原因
3. 系统自动生成预警
4. 项目经理收到通知
5. 问题解决后改回"进行中"

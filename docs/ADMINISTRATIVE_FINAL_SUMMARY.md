### 2.2 交互功能

#### 详情弹窗
1. **审批详情弹窗**
   - 显示完整审批信息
   - 物品清单、金额、用途等
   - 快速审批操作

2. **会议详情弹窗**
   - 显示会议完整信息

3. **物品详情弹窗**
   - 显示物品完整信息

#### 快速导航
- 查看全部审批按钮

- 卡片点击交互

### 2.3 数据可视化

#### 图表类型
- **柱状图**: 用于分类对比（部门、分类等）
- **饼图**: 用于占比展示（费用分类、库存分类等）
- **趋势图**: 用于时间序列数据（月度趋势）
- **进度条**: 用于完成度展示（预算使用率、库存率等）

#### 数据维度
- **时间维度**: 月度趋势、同比环比
- **分类维度**: 费用、资产、会议、请假、审批等
- **部门维度**: 部门出勤率、部门费用
- **状态维度**: 车辆状态、资产状态、审批状态

## 三、技术实现

### 3.1 技术栈
- **React 19**: 前端框架
- **Framer Motion**: 动画库
- **Tailwind CSS**: 样式框架
- **shadcn/ui**: UI 组件库
- **Lucide React**: 图标库

### 3.2 文件结构
```
frontend/src/
├── pages/

│   ├── OfficeSuppliesManagement.jsx (办公用品)
```

## 四、设计特点

### 4.1 视觉设计
- **统一风格**: 使用 shadcn/ui 设计系统
- **颜色编码**: 不同分类使用不同颜色区分
- **响应式布局**: 适配桌面、平板、移动端
- **动画效果**: 使用 Framer Motion 实现渐入动画

### 4.2 交互设计
- **悬停效果**: 卡片和图表元素悬停时高亮
- **数据标签**: 显示具体数值和百分比
- **筛选功能**: 支持按分类、状态、时间筛选
- **点击交互**: 卡片点击跳转到详情页面或弹窗

### 4.3 用户体验
- **操作流程**: 查看详情 → 弹窗显示 → 执行操作
- **信息层次**: 卡片列表 → 弹窗详情 → 详细页面
- **响应速度**: 弹窗打开/关闭动画流畅
- **视觉反馈**: 状态更新即时反馈

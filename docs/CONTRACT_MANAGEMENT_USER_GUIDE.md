## 功能概述

### 什么是合同管理模块？

合同管理模块是一个完整的合同生命周期管理系统，帮助企业：
- ✅ 规范合同创建流程
- ✅ 自动化审批流程
- ✅ 跟踪合同执行状态
- ✅ 管理合同条款和附件
- ✅ 统计分析合同数据

### 核心价值

- **规范化管理**：标准化的合同模板和流程
- **审批自动化**：根据金额自动分配审批层级
- **风险可控**：合同状态全程可追溯
- **数据可视**：实时统计合同金额和收款情况

## 合同创建

### 1. 创建销售合同

**步骤**：

1. 进入"合同管理"页面
2. 点击"新建合同"按钮
3. 填写合同基本信息：
   - 合同名称：如"XX公司自动化设备采购合同"
   - 合同类型：销售合同/采购合同/框架协议
   - 客户：选择已有客户
   - 关联商机/报价（可选）

4. 填写金额信息：
   - 合同总额：必填
   - 已收款：默认0（后续可修改）
   - 未收款：自动计算

5. 填写期限信息：
   - 签订日期
   - 生效日期
   - 到期日期
   - 合同期限（月）

6. 填写合同内容：
   - 合同标的：设备/服务描述
   - 付款条件：如"分3期付款"
   - 交付期限：如"签约后60天"

7. 指定责任人：
   - 签约销售
   - 合同管理员

8. 点击"保存"创建草稿

### 2. 合同编号规则

系统自动生成，格式：`HT-YYYYMMDD-XXX`

**示例**：
- `HT-20260215-001`：2026年2月15日第1个合同
- `HT-20260215-002`：2026年2月15日第2个合同

### 3. 添加合同条款

在合同详情页，点击"添加条款"：

**条款类型**：
- 📋 标的条款：设备清单、规格、数量
- 💰 价格条款：单价、总价、折扣
- 🚚 交付条款：交付时间、地点、方式
- 💳 付款条款：付款方式、账期、发票
- 🛡️ 质保条款：质保期、维护服务
- ⚠️ 违约条款：违约责任、赔偿标准

**示例**：
```
条款类型：付款条款
条款内容：
1. 首付30%：签约3个工作日内支付
2. 发货款40%：设备发货前支付
3. 尾款30%：验收合格后7个工作日内支付
```

### 4. 上传合同附件

支持的文件类型：
- 📄 PDF：合同文本、盖章版
- 📝 Word：合同草稿
- 📊 Excel：报价单、清单
- 📷 图片：技术方案、产品图

**上传步骤**：
1. 点击"上传附件"
2. 选择文件（单个文件<20MB）
3. 添加备注说明
4. 点击"确认上传"

## 审批流程

### 1. 审批规则

系统根据**合同金额**自动创建审批流程：

| 合同金额 | 审批流程 | 审批时长 |
|---------|---------|---------|
| < 10万 | 销售经理 | 1个工作日 |
| 10-50万 | 销售总监 | 2个工作日 |
| > 50万 | 销售总监 → 财务总监 → 总经理 | 5个工作日 |

### 2. 提交审批

**条件**：合同状态为"草稿"

**步骤**：
1. 在合同详情页检查信息完整性
2. 确认条款和附件已添加
3. 点击"提交审批"按钮
4. 填写提交说明（可选）
5. 确认提交

**提交后**：
- 合同状态变为"审批中"
- 自动通知相关审批人
- 无法再修改合同内容

### 3. 审批操作

**我的待办**：
1. 进入"待审批"列表
2. 查看合同详情
3. 选择"通过"或"驳回"
4. 填写审批意见
5. 提交审批

**审批通过**：
- 单级审批：状态变为"已审批"
- 多级审批：进入下一级审批

**审批驳回**：
- 合同状态回到"草稿"
- 创建人可修改后重新提交

### 4. 审批进度查看

在合同详情页查看"审批记录"：

```
审批层级1 - 销售总监
审批人：张总监
审批结果：✅ 通过
审批意见：同意签署
审批时间：2026-02-15 11:30

审批层级2 - 财务总监
审批人：待分配
审批结果：⏳ 待审批
审批时间：-
```

## 合同执行

### 1. 状态流转图

```
📝 草稿 
  ↓ 提交审批
⏳ 审批中
  ↓ 审批通过 / ↓ 驳回 → 草稿
✅ 已审批
  ↓ 盖章签署
✍️ 已签署
  ↓ 开始执行
🚀 执行中
  ↓ 项目交付
✔️ 已完成

❌ 作废（任意状态可作废，除"已完成"）
```

### 2. 标记为已签署

**条件**：合同状态为"已审批"

**步骤**：
1. 双方盖章签署
2. 上传盖章版合同（PDF）
3. 点击"标记为已签署"
4. 填写签署日期
5. 确认

### 3. 标记为执行中

**条件**：合同状态为"已签署"

**触发时机**：
- 项目立项
- 开始生产/交付
- 首笔款项到账

### 4. 标记为已完成

**条件**：合同状态为"执行中"

**触发时机**：
- 项目验收通过
- 款项全部到账
- 交付完成

### 5. 作废合同

**适用场景**：
- 客户取消订单
- 合同谈判失败
- 重大变更需重签

**操作**：
1. 点击"作废合同"
2. 填写作废原因
3. 确认作废

⚠️ **注意**：
- 已完成的合同不能作废
- 作废后合同不可恢复
- 建议先咨询法务部门

## 条款管理

### 1. 条款模板库

系统提供常用条款模板：

**付款条款模板**：
```
模板1：全款预付
合同签订后3个工作日内支付全款

模板2：标准三期
首付30%（签约后）、发货款40%（发货前）、尾款30%（验收后）

模板3：月结账期
月结30天，每月5日前支付上月发生额
```

**质保条款模板**：
```
模板1：标准质保
质保期12个月，免费维护和零配件更换

模板2：延长质保
质保期24个月，第一年免费，第二年有偿服务

模板3：终身质保
终身提供维护服务，免费保养，零配件按成本价
```

### 2. 自定义条款

可根据实际情况编辑条款内容：

**编辑技巧**：
- 使用编号清晰列举
- 明确时间节点
- 量化标准（金额、数量、期限）
- 附加违约条款

## 附件管理

### 1. 附件分类

建议按类型组织附件：

- 📄 **合同正本**：盖章版PDF
- 📝 **合同草稿**：Word版（修订痕迹）
- 📊 **报价单**：详细报价表
- 🔧 **技术方案**：设备清单、参数
- 📷 **产品图片**：设备外观、功能图
- 📋 **其他附件**：补充协议、往来邮件

### 2. 文件命名规范

建议格式：`合同编号-文件类型-版本.扩展名`

**示例**：
```
HT-20260215-001-合同正本-V1.0.pdf
HT-20260215-001-报价单-V2.0.xlsx
HT-20260215-001-技术方案-最终版.pdf
```

### 3. 版本管理

对于多次修订的文件：
- 每次修改上传新版本
- 在文件名中标注版本号
- 保留历史版本备查
- 删除时需谨慎

## 常见问题

### Q1：合同编号能否自定义？

**A**：创建时可以指定自定义编号，但建议使用系统自动生成的编号，确保唯一性和规范性。

### Q2：合同审批能否跳级？

**A**：不可以。必须按照设定的审批流程逐级审批。如需调整，请联系管理员修改审批规则。

### Q3：审批驳回后能否直接签署？

**A**：不可以。驳回后状态回到"草稿"，需要修改后重新提交审批。

### Q4：已签署的合同能否修改金额？

**A**：不可以直接修改。如需变更，应：
1. 使用"合同变更"功能
2. 或作废原合同，重新签订

### Q5：删除合同后能否恢复？

**A**：无法恢复。删除前请确认，建议使用"作废"代替删除。

### Q6：附件大小限制是多少？

**A**：单个文件最大20MB。如有大文件，建议使用云盘链接。

### Q7：如何批量导出合同？

**A**：
1. 进入合同列表
2. 勾选需要导出的合同
3. 点击"批量导出"
4. 选择导出格式（Excel/PDF）

### Q8：合同到期后如何续签？

**A**：
1. 在原合同基础上"复制创建"
2. 修改合同期限
3. 重新提交审批流程

### Q9：如何查看历史审批记录？

**A**：在合同详情页点击"审批记录"标签，查看完整审批历史。

### Q10：收款记录如何更新？

**A**：
1. 进入合同详情
2. 点击"收款记录"
3. 添加收款记录（金额、日期、凭证）
4. 系统自动更新"已收款"和"未收款"金额

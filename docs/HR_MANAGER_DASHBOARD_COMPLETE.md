## 📝 使用说明

### 查看员工详情
1. 进入"员工管理"Tab
2. 在员工列表中找到目标员工
3. 点击"查看"按钮（眼睛图标）
4. 员工详情对话框将显示完整信息

### 刷新数据
1. 点击页面头部右上角的"刷新"按钮
2. 按钮图标会旋转，表示正在刷新
3. 刷新完成后，数据自动更新

### 快捷操作
1. 在"概览"Tab中
2. 点击快捷操作卡片的任意按钮
3. 自动跳转到对应的功能Tab

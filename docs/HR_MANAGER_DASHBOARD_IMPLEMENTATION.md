## 🎨 设计规范

### 颜色方案
- **员工统计**: 蓝色 (blue-400)
- **招聘管理**: 琥珀色 (amber-400)
- **绩效管理**: 紫色 (purple-400)
- **考勤管理**: 绿色 (emerald-400)
- **员工关系**: 粉色 (pink-400)
- **绩效分数**: 青色 (cyan-400)

### 状态颜色
- **正常/成功**: 绿色 (emerald)
- **警告/待处理**: 琥珀色 (amber)
- **紧急/错误**: 红色 (red)
- **进行中**: 蓝色 (blue)

## 📝 使用说明

### 访问路径
1. 使用 `hr_manager` 角色登录
2. 系统自动跳转到 `/hr-manager-dashboard`
3. 或通过侧边栏导航 → 人事管理 → 人事工作台

### 主要功能入口
- **概览**: 默认Tab，显示关键信息和待办事项
- **招聘管理**: 查看和管理招聘需求
- **绩效管理**: 查看和管理绩效评审
- **考勤管理**: 查看考勤统计
- **员工管理**: 查看和管理员工信息
- **员工关系**: 处理员工关系问题

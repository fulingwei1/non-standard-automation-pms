## 🎯 系统概述

### 什么是售前AI系统？

售前AI系统是一套智能化的售前工作辅助平台，通过人工智能技术帮助售前团队：

✅ **自动理解客户需求** - AI快速分析客户需求文档  
✅ **智能生成技术方案** - 基于需求自动生成初步方案  
✅ **精准成本估算** - AI辅助项目成本评估  
✅ **预测项目赢率** - 分析项目成功概率  
✅ **快速生成报价** - 自动生成标准化报价文档

## 🚀 功能介绍

### 1. AI仪表盘 📊

**位置**: 主菜单 → 售前AI → AI仪表盘

**功能说明**:
- 查看AI系统整体运行状况
- 实时监控AI使用统计
- 分析AI功能使用趋势
- 查看团队使用情况

**关键指标**:
- 总使用次数
- AI成功率
- 平均响应时间
- 活跃用户数

### 2. AI工作台 💼

**位置**: 主菜单 → 售前AI → AI工作台

**功能说明**:
- 管理所有售前工单
- 启动AI工作流
- 跟踪AI处理进度
- 查看AI处理结果

**工作流程**:
```
新建工单 → 填写基本信息 → 启动AI工作流 → AI自动处理 → 查看结果 → 人工审核 → 生成报价
```

### 3. AI功能模块 🤖

#### 3.1 需求理解AI
- **功能**: 自动分析客户需求文档
- **输入**: 客户需求描述、附件文档
- **输出**: 结构化需求分析报告
- **应用场景**: 快速理解复杂需求

#### 3.2 方案生成AI
- **功能**: 基于需求自动生成技术方案
- **输入**: 需求分析结果
- **输出**: 初步技术方案文档
- **应用场景**: 标准化方案快速生成

#### 3.3 成本估算AI
- **功能**: 智能评估项目成本
- **输入**: 技术方案、工作量估算
- **输出**: 成本明细表
- **应用场景**: 快速成本核算

#### 3.4 赢率预测AI
- **功能**: 预测项目成功概率
- **输入**: 项目信息、历史数据
- **输出**: 赢率分数、风险分析
- **应用场景**: 项目决策支持

#### 3.5 报价生成AI
- **功能**: 自动生成正式报价文档
- **输入**: 成本明细、利润率
- **输出**: 标准化报价文档
- **应用场景**: 快速生成规范报价

## 🎓 快速上手

### 第一步：登录系统

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮

### 第二步：创建售前工单

1. 点击主菜单 → **售前AI** → **AI工作台**
2. 点击右上角 **"新建工单"** 按钮
3. 填写工单基本信息：
   - 工单标题
   - 客户名称
   - 需求描述
   - 附件上传（可选）
4. 点击 **"保存"** 按钮

### 第三步：启动AI工作流

1. 在工单列表中找到刚创建的工单
2. 点击工单行末的 **"启动AI"** 按钮
3. 确认启动对话框
4. AI工作流自动开始处理

### 第四步：查看AI处理结果

1. 点击工单标题，进入工单详情页
2. 查看右侧 **"AI工作流进度"** 面板
3. 等待AI处理完成（各步骤显示绿色✓）
4. 查看每个步骤的AI处理结果

### 第五步：人工审核与调整

1. 在工单详情页，查看AI生成的内容
2. 对AI结果进行人工审核
3. 必要时进行修改和调整
4. 点击 **"提交反馈"** 帮助AI改进

### 第六步：生成最终报价

1. 确认所有信息无误
2. 点击 **"生成报价"** 按钮
3. 下载PDF/Word格式的报价文档
4. 发送给客户

## 📖 详细操作指南

### 如何使用需求理解AI？

**场景**: 客户发来一份50页的需求文档，需要快速理解核心需求

**操作步骤**:

1. 在工单详情页，找到 **"需求理解AI"** 面板
2. 上传客户需求文档（支持PDF、Word、TXT）
3. 或直接粘贴需求文本到输入框
4. 点击 **"开始分析"** 按钮
5. 等待AI处理（通常10-30秒）
6. 查看分析结果：
   - 需求摘要
   - 核心功能点
   - 技术要求
   - 非功能性需求
   - 风险点识别

**小贴士**:
- 📝 文档越清晰，AI分析越准确
- 🔄 可以多次运行，调整输入
- 💡 结合人工经验，不要完全依赖AI

### 如何使用方案生成AI？

**场景**: 需要快速出一份初步技术方案

**操作步骤**:

1. 确保需求理解AI已完成
2. 在 **"方案生成AI"** 面板中
3. 选择方案模板（可选）：
   - 标准Web应用方案
   - 移动App方案
   - 系统集成方案
   - 定制开发方案
4. 配置方案参数：
   - 技术栈偏好
   - 架构模式
   - 部署方式
5. 点击 **"生成方案"** 按钮
6. 等待AI生成（通常30-60秒）
7. 查看方案内容：
   - 系统架构图
   - 技术选型
   - 功能模块设计
   - 实施计划

**小贴士**:
- 🎨 可以选择不同模板多次生成对比
- ✏️ 生成后可以直接编辑修改
- 📎 支持导出Word/PDF格式

### 如何使用成本估算AI？

**场景**: 需要快速评估项目成本

**操作步骤**:

1. 确保技术方案已生成
2. 在 **"成本估算AI"** 面板中
3. 输入基本信息：
   - 项目周期（月）
   - 团队规模（人）
   - 技术复杂度（低/中/高）
4. 或者让AI自动从方案中提取信息
5. 点击 **"开始估算"** 按钮
6. 查看成本明细：
   - 人力成本
   - 硬件成本
   - 软件许可成本
   - 其他成本
   - 总成本

**小贴士**:
- 💰 AI估算作为参考，需人工复核
- 📊 可以调整各项参数重新计算
- 📈 查看成本分布饼图

### 如何使用赢率预测AI？

**场景**: 评估项目成功概率，辅助决策

**操作步骤**:

1. 在 **"赢率预测AI"** 面板中
2. AI自动分析项目因素：
   - 客户历史合作情况
   - 竞争对手情况
   - 预算匹配度
   - 技术可行性
   - 团队经验
3. 点击 **"预测赢率"** 按钮
4. 查看预测结果：
   - 赢率分数（0-100）
   - 影响因素分析
   - 风险点提示
   - 改进建议

**小贴士**:
- 🎯 高于70分建议积极跟进
- ⚠️ 低于40分需谨慎评估
- 📋 查看详细分析报告

### 如何使用报价生成AI？

**场景**: 快速生成标准化报价文档

**操作步骤**:

1. 确保成本估算已完成
2. 在 **"报价生成AI"** 面板中
3. 设置报价参数：
   - 利润率（%）
   - 付款方式
   - 交付条款
   - 报价有效期
4. 选择报价模板
5. 点击 **"生成报价"** 按钮
6. 预览报价文档
7. 点击 **"导出"** 选择格式：
   - PDF（推荐）
   - Word（可编辑）
   - Excel（成本明细）

**小贴士**:
- 📄 生成前可预览效果
- ✨ 支持自定义公司logo
- 🔒 可设置报价文档密码

## ❓ 常见问题

### Q1: AI处理失败怎么办？

**A**: 如果AI处理失败：
1. 查看错误信息提示
2. 检查输入数据是否完整
3. 尝试重新运行
4. 联系技术支持

### Q2: AI生成的内容不准确怎么办？

**A**: AI生成的内容仅供参考：
1. 进行人工审核和修改
2. 提交反馈帮助AI改进
3. 结合人工经验完善内容

### Q3: 如何提高AI处理速度？

**A**: 
1. 输入清晰简洁的信息
2. 避免上传过大的附件
3. 在系统负载低时使用

### Q4: 可以批量处理工单吗？

**A**: 
1. 可以，在工单列表页
2. 选择多个工单
3. 点击 **"批量处理"** 按钮
4. 选择要执行的AI功能

### Q5: AI处理记录保存多久？

**A**: 
- 所有AI处理记录永久保存
- 可以随时查看历史记录
- 支持导出为报告

## 💡 最佳实践

### 1. 工单信息填写要规范

- ✅ 使用清晰的工单标题
- ✅ 详细描述客户需求
- ✅ 上传相关文档
- ✅ 标注重要信息

### 2. 合理使用AI功能

- ✅ AI作为辅助工具，不是完全替代
- ✅ AI结果需要人工审核
- ✅ 复杂项目建议分步骤处理

```
接收需求 → 创建工单 → AI初步分析 → 人工审核 → AI深化方案 →

```

- 🔔 设置关键节点提醒

## 五、备注

1. **销售管理模块**：在销售工程师菜单中，包含大量销售相关功能，报价管理只是其中一部分
2. **ECN管理**：目前是独立路由，没有明确的模块归属
3. **进度跟踪**：功能分散在项目管理和项目详情页面中
4. **成本管理**：功能分散在销售、采购、财务等多个模块中

## 🚀 使用方法

### 1. **登录页面快捷登录**
- 打开应用首页
- 在"快捷登录"区域直接点击对应角色按钮
- 自动填充用户名和密码
- 点击"登录"按钮

**快捷登录区域 (2行 x 3.5列):**
```
┌──────────────────────────────────────────┐
│     快捷登录（7个演示账户）               │
├──────────────────────────────────────────┤
│  │ 董事长   │ 总经理   │ 项目经理    │
│  │chairman  │ gm       │ pm         │
├──────────────────────────────────────────┤
│  │ 销售总监 │ 销售经理 │ P&C │机械部经理│
│  │sales_dir │sales_mgr │ pmc │mech_mgr  │
└──────────────────────────────────────────┘
```

### 2. **手动登录**
- 在用户名字段输入账户名
- 在密码字段输入密码
- 点击"登录"按钮

### 3. **首页角色导航**
登录后在首页会看到 **"快速导航"** 区域，显示 7 张角色卡片：

```
┌────────────────────────────────────────────────────┐
│                  快速导航                           │
├────────────────────────────────────────────────────┤
│  [👑董事长]  [⚡总经理]  [🔀项目经理]               │
│  [📈销售总监] [💼销售经理] [🛒P&C] [🔧机械部经理]   │
└────────────────────────────────────────────────────┘
```

### 4. **点击卡片快速跳转**
- 每个卡片代表一个角色的专属工作区
- 点击卡片即可跳转到对应的仪表盘

## 📝 注意事项

1. **演示账户密码**: 为了便利演示，所有账户密码都设置为 `123456`
2. **生产环境**: 请勿在生产环境中使用这些演示账户
3. **数据隔离**: 各角色间的数据访问权限已通过后端权限系统进行隔离
4. **销售经理角色映射**: 演示账户展示为"销售经理"，但在系统中使用真实的 SA (销售专员) 角色。这确保所有演示账户都使用真实存在的系统角色
5. **快捷登录位置**: 登录页面底部的快捷登录区域便于快速演示

## 六、设计特点

### 6.1 视觉设计
- **统一风格**: 使用 shadcn/ui 设计系统
- **颜色编码**: 不同分类使用不同颜色区分
- **响应式布局**: 适配桌面、平板、移动端
- **动画效果**: 使用 Framer Motion 实现渐入动画

### 6.2 交互设计
- **悬停效果**: 卡片和图表元素悬停时高亮
- **数据标签**: 显示具体数值和百分比
- **筛选功能**: 支持按分类、状态、时间筛选
- **点击交互**: 卡片点击跳转到详情页面

### 6.3 数据可视化
- **多种图表类型**: 柱状图、饼图、趋势图、进度条
- **趋势指示**: 使用箭头和颜色表示趋势方向
- **占比展示**: 清晰的占比和进度可视化
- **对比分析**: 支持多维度对比

### 7.1 技术栈
- **React 19**: 前端框架

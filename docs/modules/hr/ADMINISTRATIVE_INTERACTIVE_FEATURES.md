# 行政经理工作台交互功能增强

## 一、新增交互功能

### 1.1 详情弹窗功能

#### 审批详情弹窗
- **触发方式**: 点击待审批事项卡片
- **功能**:
  - 显示完整的审批信息
  - 物品清单展示（办公用品申请）
  - 申请金额显示
  - 用途说明
  - 会议室信息（会议申请）
  - 请假天数（请假申请）
  - 快速审批操作（批准/拒绝按钮）

#### 会议详情弹窗
- **触发方式**: 点击今日会议卡片
- **功能**:
  - 显示会议完整信息
  - 会议室、时间、日期、参会人数
  - 会议状态标识
  - 查看完整信息按钮

### 1.2 快速导航功能

#### 查看全部审批
- **位置**: 待审批事项卡片底部
- **功能**: 点击跳转到审批中心页面 (`/administrative-approvals`)

#### 详情按钮
- **位置**: 各功能模块标题栏
- **功能**: 快速跳转到对应的详细管理页面

## 三、技术实现

### 3.1 状态管理
```javascript
const [selectedApproval, setSelectedApproval] = useState(null)
const [showApprovalDetail, setShowApprovalDetail] = useState(false)
const [selectedMeeting, setSelectedMeeting] = useState(null)
const [showMeetingDetail, setShowMeetingDetail] = useState(false)
```

### 3.2 Dialog组件
- 使用 shadcn/ui 的 Dialog 组件
- 支持打开/关闭状态控制
- 响应式设计，适配不同屏幕

### 3.3 事件处理
```javascript
// 点击审批卡片
onClick={() => {
  setSelectedApproval(item)
  setShowApprovalDetail(true)
}}

// 点击会议卡片
onClick={() => {
  setSelectedMeeting(meeting)
  setShowMeetingDetail(true)
}}
```

## 五、用户体验提升

### 5.1 操作流程优化
1. **查看详情**: 点击卡片 → 弹窗显示 → 查看信息

# 行政经理工作台详细设计文档

## 一、概述

本文档详细描述了行政经理登录后的工作台页面设计，包括功能模块、数据展示、交互设计等。

## 二、角色定义

### 2.1 角色信息
- **角色代码**: `administrative_manager`
- **角色名称**: 行政经理
- **数据范围**: 全部（`all`）
- **主要职责**: 
  - 行政事务管理
  - 办公用品管理
  - 会议管理
  - 车辆管理
  - 固定资产管理
  - 员工考勤管理
  - 行政审批流程

### 2.2 导航配置
行政经理拥有独立的导航组 `administrative`，包含以下功能模块：
- 行政工作台（主页面）
- 办公用品管理
- 会议管理
- 车辆管理
- 固定资产
- 员工考勤
- 请假管理
- 审批中心
- 费用统计

## 四、功能模块详细设计

### 4.1 关键指标卡片区

#### 4.1.1 待审批事项
- **图标**: ClipboardCheck
- **颜色**: 红色系（text-red-400）
- **主要数据**: 
  - 待审批总数
  - 紧急事项数量
- **交互**: 点击跳转到审批中心

#### 4.1.2 低库存物品
- **图标**: AlertTriangle
- **颜色**: 琥珀色系（text-amber-400）
- **主要数据**: 
  - 低库存物品数量
  - 需要补货提示
- **交互**: 点击跳转到办公用品管理

#### 4.1.3 今日会议
- **图标**: Calendar
- **颜色**: 蓝色系（text-blue-400）
- **主要数据**: 
  - 今日会议场次
  - 本周会议总数
- **交互**: 点击跳转到会议管理

#### 4.1.4 在用车辆
- **图标**: Car
- **颜色**: 青色系（text-cyan-400）
- **主要数据**: 
  - 当前使用中车辆数
  - 车辆总数
- **交互**: 点击跳转到车辆管理

#### 4.1.5 员工出勤率
- **图标**: UserCheck
- **颜色**: 绿色系（text-emerald-400）
- **主要数据**: 
  - 本月平均出勤率
  - 百分比显示
- **交互**: 点击跳转到员工考勤

#### 4.1.6 固定资产
- **图标**: Building2
- **颜色**: 紫色系（text-purple-400）
- **主要数据**: 
  - 固定资产总数
  - 固定资产总值
- **交互**: 点击跳转到固定资产管理

### 4.2 月度行政预算执行

#### 4.2.1 预算概览
- **预算总额**: 显示月度行政预算总金额
- **已使用**: 显示已使用的预算金额（琥珀色高亮）
- **使用率进度条**: 可视化显示预算使用率
- **剩余预算**: 显示剩余可用预算

#### 4.2.2 费用分类统计
- **办公用品费用**: 显示办公用品相关支出
- **车辆费用**: 显示车辆相关支出（燃油、保养等）
- **其他费用**: 显示其他行政支出

### 4.3 办公用品库存预警

#### 4.3.1 物品列表
每个物品显示：
- **物品名称**: 如"A4打印纸"、"黑色签字笔"
- **分类标签**: 办公耗材、办公文具等
- **库存状态**: 
  - 正常（绿色）
  - 库存不足（红色警告）
- **当前库存**: 显示当前库存数量和单位
- **最低库存**: 显示最低库存阈值
- **供应商信息**: 显示供应商名称
- **库存率进度条**: 可视化显示库存率

#### 4.3.2 交互功能
- 点击"查看全部"跳转到办公用品管理页面
- 点击物品卡片可查看详细信息

### 4.4 今日会议安排

#### 4.4.1 会议卡片
每个会议显示：
- **会议标题**: 如"项目进度评审会"
- **会议状态**: 
  - 进行中（绿色）
  - 已安排（蓝色）
- **组织者信息**: 组织者姓名和部门
- **会议详情**: 
  - 会议室位置（图标+文字）
  - 会议时间（图标+时间范围）
  - 参会人数（图标+人数）

#### 4.4.2 交互功能
- 点击"查看全部"跳转到会议管理页面
- 点击会议卡片可查看会议详情

### 4.5 待审批事项（右侧边栏）

#### 4.5.1 审批项列表
每个审批项显示：
- **审批类型标签**: 
  - 办公用品（蓝色）
  - 车辆（青色）
  - 资产（紫色）
  - 会议（绿色）
  - 请假（粉色）
- **紧急标识**: 高优先级显示红色"紧急"标签
- **审批标题**: 如"办公用品采购申请"
- **申请人信息**: 部门和申请人姓名
- **详细信息**: 
  - 办公用品：物品列表
  - 车辆：使用目的和目的地
  - 资产：资产名称和金额
  - 会议：会议室、日期、时间
  - 请假：请假类型、天数、日期
- **金额信息**: 如有金额，显示在底部
- **提交时间**: 显示提交的具体时间

#### 4.5.2 交互功能
- 点击审批项可查看详情并处理
- 点击"查看全部审批"跳转到审批中心

### 4.6 车辆使用状态（右侧边栏）

#### 4.6.1 车辆卡片
每辆车显示：
- **车牌号**: 如"粤B12345"
- **车辆品牌**: 如"丰田凯美瑞"
- **使用状态**: 
  - 使用中（蓝色）
  - 可用（绿色）
  - 保养中（琥珀色）
- **详细信息**:
  - 使用中：驾驶员、使用目的、目的地、使用时间
  - 可用：下次保养日期
  - 保养中：保养原因、预计归还日期

#### 4.6.2 交互功能
- 点击"详情"跳转到车辆管理页面
- 点击车辆卡片可查看详细信息

### 4.7 部门出勤统计（右侧边栏）

#### 4.7.1 部门统计卡片
每个部门显示：
- **部门名称**: 如"销售部"、"项目部"
- **出勤数据**: 
  - 总人数
  - 出勤人数
  - 请假人数（如有，琥珀色显示）
  - 迟到人数（如有，红色显示）
- **出勤率**: 大号绿色数字显示
- **出勤率进度条**: 可视化显示出勤率

#### 4.7.2 交互功能
- 点击"详情"跳转到员工考勤页面
- 点击部门卡片可查看详细出勤记录

## 六、交互设计

### 6.1 页面加载
- 使用 Framer Motion 实现渐入动画
- 卡片使用 stagger 动画效果，依次出现
- 数据加载时显示骨架屏

### 6.2 卡片交互
- **悬停效果**: 卡片边框颜色加深，阴影增强
- **点击交互**: 
  - 统计卡片点击跳转到对应功能页面
  - 列表项点击查看详情或处理
- **状态反馈**: 使用颜色和图标清晰标识状态

### 6.3 数据刷新
- 关键数据支持实时刷新
- 提供手动刷新按钮
- 数据更新时使用动画提示

## 七、视觉设计

### 7.1 颜色系统
- **主色调**: 深色背景（slate-800/900）
- **强调色**: 
  - 红色：紧急/警告（审批、低库存）
  - 琥珀色：注意（预算使用）
  - 蓝色：信息（会议、办公用品）
  - 青色：车辆相关
  - 绿色：正常/成功（出勤、可用）
  - 紫色：资产相关

### 7.2 图标系统
- 使用 Lucide React 图标库
- 图标大小统一为 5x5 (h-5 w-5)
- 图标颜色与功能模块颜色一致

### 7.3 卡片设计
- 圆角：rounded-lg (0.5rem)
- 边框：border-slate-700/50
- 背景：渐变背景 from-slate-800/50 to-slate-900/50
- 悬停：边框颜色加深，阴影增强

### 7.4 进度条设计
- 高度：h-4（主要进度条）或 h-1.5（次要进度条）
- 背景：bg-slate-700/50
- 进度：渐变 from-violet-500 to-indigo-500

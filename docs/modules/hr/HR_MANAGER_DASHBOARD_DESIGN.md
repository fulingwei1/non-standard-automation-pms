# 人事经理工作台 - 详细设计文档

## 一、页面概述

### 1.1 页面定位
人事经理工作台是人事经理登录后的主页面，提供人力资源管理的核心功能入口和数据概览。

### 1.2 核心职责
- **人力资源规划**：员工总数、部门分布、人员结构分析
- **招聘管理**：招聘需求、招聘进度、面试安排、录用审批
- **绩效管理**：绩效评审、绩效分布、绩效分析
- **员工关系**：员工问题处理、员工满意度、关系维护
- **考勤管理**：出勤统计、考勤异常、请假审批

### 1.3 数据范围
- **数据权限**：`all`（全部数据）
- **可见范围**：全公司所有员工、部门、招聘、绩效数据

## 三、功能模块详细设计

### 3.1 概览 Tab（Overview）

#### 3.1.1 左侧区域（2/3宽度）

**待处理招聘卡片**
- 显示待处理的招聘需求列表
- 每个招聘项显示：
  - 职位名称、部门、创建日期
  - 紧急程度标签（紧急/普通）
  - 状态标签（招聘中/筛选中/面试中/发Offer）
  - 预算金额
  - 申请人数量、面试人数
  - 期望入职日期
- 底部"查看全部招聘"按钮

**待绩效评审卡片**
- 显示待完成的绩效评审列表
- 每个评审项显示：
  - 员工姓名、部门
  - 考核周期（如：2024年Q4）
  - 截止日期
  - 优先级标签（紧急/普通）
  - 状态标签（待处理/评审中）
- 底部"查看全部评审"按钮

#### 3.1.2 右侧区域（1/3宽度）

**员工概览卡片**
- 在职率：进度条显示在职员工占比
- 本月新增：显示新增员工数量（绿色）
- 本月离职：显示离职员工数量（红色）
- 平均工龄：显示平均工作年限

**绩效分布卡片**
- 饼图形式显示绩效等级分布：
  - 优秀（绿色）：18.0%
  - 良好（蓝色）：47.8%
  - 合格（黄色）：27.0%
  - 待改进（红色）：7.3%
- 每个等级显示人数和百分比

#### 3.1.3 部门人员分布卡片（全宽）

- 网格布局显示各部门人员信息
- 每个部门卡片显示：
  - 部门名称
  - 新增/离职标签（如有）
  - 在职人数/总人数
  - 平均绩效分
  - 平均年龄、平均工龄
  - 在职率进度条

### 3.2 招聘管理 Tab（Recruitment）

#### 3.2.1 统计卡片（4个）
- **进行中招聘**：显示正在进行的招聘数量
- **已完成招聘**：显示本月已完成的招聘数量
- **招聘成功率**：显示招聘成功率百分比
- **待审批**：显示待审批的招聘需求数量

#### 3.2.2 招聘趋势图表
- 显示最近6个月的招聘趋势
- 每个月份显示：
  - 发布的职位数量
  - 成功录用的人数
  - 录用率进度条

### 3.3 绩效管理 Tab（Performance）

#### 3.3.1 统计卡片（4个）
- **待评审**：显示待完成的绩效评审数量
- **已完成**：显示已完成的绩效评审数量
- **完成率**：显示绩效评审完成率百分比
- **平均分**：显示全公司平均绩效分数

#### 3.3.2 功能说明
- 后续可扩展：绩效分布图表、部门绩效对比、绩效趋势分析等

### 3.4 考勤管理 Tab（Attendance）

#### 3.4.1 统计卡片（4个）
- **今日出勤率**：显示今日出勤率百分比和进度条
- **月度出勤率**：显示本月出勤率百分比和进度条
- **迟到人数**：显示今日迟到人数
- **缺勤人数**：显示今日缺勤人数

#### 3.4.2 功能说明
- 后续可扩展：考勤明细、请假统计、异常考勤处理等

### 3.5 员工管理 Tab（Employees）

#### 3.5.1 功能说明
- 当前显示"员工管理功能开发中..."
- 后续可扩展：员工列表、员工详情、员工档案管理等

### 3.6 员工关系 Tab（Relations）

#### 3.6.1 统计卡片（3个）
- **待处理问题**：显示待处理的员工关系问题数量
- **已解决问题**：显示已解决的员工关系问题数量
- **员工满意度**：显示员工满意度百分比

#### 3.6.2 员工关系问题列表
- 显示员工关系问题列表
- 每个问题显示：
  - 问题类型标签（冲突/请假/投诉/绩效）
  - 问题标题
  - 问题描述
  - 员工姓名、部门
  - 创建日期
  - 优先级标签（紧急/普通）
  - 状态标签（待处理/评审中/处理中）

## 五、交互设计

### 5.1 页面加载
- 使用 `framer-motion` 实现渐入动画
- 统计卡片使用 `stagger` 效果依次显示

### 5.2 卡片交互
- 所有卡片支持 hover 效果（边框高亮、阴影）
- 可点击的卡片显示 cursor-pointer
- 列表项支持点击跳转到详情页

### 5.3 Tab 切换
- 使用 `Tabs` 组件实现平滑切换
- 切换时保持数据状态

### 5.4 快捷操作
- 页面头部提供"新建招聘"和"绩效管理"快捷按钮
- 按钮使用主色调，突出显示

## 十、设计原则

### 10.1 信息优先级
1. **关键指标**：在职员工、待审批事项、出勤率等核心数据
2. **待办事项**：待处理招聘、待绩效评审等需要立即处理的事项
3. **统计分析**：部门分布、绩效分布等分析数据

### 10.2 用户体验
- **一目了然**：关键数据在页面顶部清晰展示
- **快速操作**：提供快捷按钮，减少操作步骤
- **信息分层**：使用 Tab 组织不同功能模块
- **视觉反馈**：使用颜色、图标、进度条等提供视觉反馈

### 10.3 响应式设计
- 支持桌面端、平板端、移动端
- 使用 Tailwind 响应式类名实现适配
- 统计卡片在小屏幕下自动调整为单列布局

## 十一、总结

人事经理工作台是一个功能完整、信息丰富的管理页面，为人事经理提供了：

1. **全面的数据概览**：通过6个关键统计卡片快速了解人力资源状况
2. **清晰的功能组织**：通过 Tab 标签页组织不同功能模块
3. **高效的待办管理**：突出显示待处理的招聘和绩效评审
4. **直观的数据可视化**：使用进度条、图表等展示数据分布
5. **良好的用户体验**：流畅的动画、清晰的视觉层次、便捷的操作

该页面设计遵循了系统的整体设计风格，与其他角色的工作台保持一致，同时针对人事管理的特殊需求进行了优化。

## 📝 使用说明

### 访问路径
1. 使用 `hr_manager` 角色登录
2. 系统自动跳转到 `/hr-manager-dashboard`
3. 或通过侧边栏导航 → 人事管理 → 人事工作台

### 主要功能入口
- **快捷操作**: 概览Tab中的6个快捷按钮
- **页面头部**: 新建招聘、绩效管理、刷新、更多操作
- **Tab导航**: 7个功能Tab切换
- **统计周期**: 统计分析Tab中的周期切换

### 操作流程
1. **查看概览**: 默认Tab，查看关键信息和待办事项
2. **快捷操作**: 点击快捷操作卡片快速跳转到对应Tab
3. **详细管理**: 在各个Tab中查看和管理详细信息
4. **统计分析**: 在统计分析Tab中查看多维度数据

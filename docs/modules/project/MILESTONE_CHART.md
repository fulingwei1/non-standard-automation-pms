# 短期里程碑甘特概览

> 以 6 周周期展示 M0-M6 的时间跨度与关键交付，用于汇报或打印张贴。

```
周次 →    W0   W1   W2   W3   W4   W5   W6

M0 启动  [■■]
M1 调度治理        [■■■■]
M2 Scheduler 回归       [■■■■]
M3 前端/DB 联调             [■■■■]
M4 UAT Checkpoint                [■■■■]
M5 数据字典评审                    [■■■]
M6 发布审批                           [■■]
```

## 里程碑说明

- **M0 (W0)**：Kickoff，锁定人力和 SP，输出 `SHORT_TERM_ACTION_BOARD` 初稿。
- **M1 (W1)**：调度治理方案明确，完成健康度/定时服务盘点；更新燃尽计划。
- **M2 (W2)**：Scheduler 回归通过，预警通知链路验证完成，形成回归报告。
- **M3 (W3)**：生产/验收页面 Demo 与 DB 迁移初稿评审，确认联调结果。
- **M4 (W4)**：UAT 检查点，移动端样例与缺料/研发表数据导入完成。
- **M5 (W5)**：数据字典评审，研发项目 6 张表落地并同步字段说明。
- **M6 (W6)**：发布审批，完成燃尽复盘与上线材料。

> 颜色深浅可在 PPT 或 Miro 中进一步美化，此 ASCII 版用于文本场景。

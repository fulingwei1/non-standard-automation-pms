# OEE计算说明文档

## 1. OEE概述

OEE (Overall Equipment Effectiveness,设备综合效率) 是衡量制造业生产效率的国际标准指标,由Seiichi Nakajima在TPM(Total Productive Maintenance)理论中提出。

### 1.1 OEE的重要性

- **全面性**: 综合考虑设备可用性、性能和质量三个维度
- **标准性**: 国际通用标准,便于行业对标
- **可操作性**: 明确指出改进方向(可用率/性能率/质量率)
- **量化性**: 可量化跟踪改进效果

### 1.2 世界级OEE标准

- **世界级**: OEE ≥ 85%
  - 可用率 ≥ 90%
  - 性能率 ≥ 95%
  - 合格率 ≥ 99%

- **良好**: OEE 60% - 85%
- **需改进**: OEE < 60%

## 2. OEE计算公式(国际标准)

### 2.1 核心公式

```
OEE = 可用率(Availability) × 性能率(Performance) × 合格率(Quality)
```

### 2.2 三要素详细计算

#### 2.2.1 可用率 (Availability)

**定义**: 设备实际运行时间占计划生产时间的比例

```
可用率 = 运行时间 / 计划生产时间 × 100%

其中:
运行时间 = 计划生产时间 - 计划停机时间 - 非计划停机时间

计划生产时间: 班次总时间 - 计划休息时间
计划停机时间: 计划内的保养、换模、调试等
非计划停机时间: 故障、待料、调整等
```

**示例**:
```
计划生产时间: 8小时 = 480分钟
计划停机(保养): 30分钟
非计划停机(故障): 20分钟

运行时间 = 480 - 30 - 20 = 430分钟
可用率 = 430 / 480 × 100% = 89.58%
```

**影响因素**:
- 设备故障
- 换模时间
- 缺料待工
- 设备调整
- 计划保养

#### 2.2.2 性能率 (Performance)

**定义**: 设备实际生产速度与理想生产速度的比较

```
性能率 = (理想周期时间 × 实际产量) / 运行时间 × 100%

或

性能率 = 实际产量 / 目标产量 × 100%

注意: 性能率上限为100%(如果实际超过理想,应检查数据准确性)
```

**示例**:
```
理想周期时间: 2分钟/件
实际产量: 200件
运行时间: 430分钟

性能率 = (2 × 200) / 430 × 100% = 93.02%
```

**影响因素**:
- 设备运行速度
- 微停机
- 速度损失
- 空转浪费

#### 2.2.3 合格率 (Quality)

**定义**: 合格品占总产量的比例

```
合格率 = 合格数量 / 实际产量 × 100%

或

合格率 = (实际产量 - 不良品数 - 返工数) / 实际产量 × 100%
```

**示例**:
```
实际产量: 200件
合格数量: 190件
不良品: 10件

合格率 = 190 / 200 × 100% = 95%
```

**影响因素**:
- 质量控制
- 原料质量
- 操作技能
- 工艺参数

### 2.3 完整计算示例

**场景**: 某设备8小时班次生产记录

```
输入数据:
- 计划生产时间: 480分钟(8小时)
- 计划停机(保养): 30分钟
- 非计划停机(故障): 20分钟
- 理想周期时间: 2分钟/件
- 实际产量: 200件
- 合格数量: 190件

计算过程:
1. 运行时间 = 480 - 30 - 20 = 430分钟

2. 可用率 = 430 / 480 × 100% = 89.58%

3. 性能率 = (2 × 200) / 430 × 100% = 93.02%

4. 合格率 = 190 / 200 × 100% = 95%

5. OEE = 89.58% × 93.02% × 95% = 79.14%
```

**结论**: OEE为79.14%,处于"良好"水平,有提升空间

## 3. 六大损失分析

OEE背后是对六大损失的量化管理:

### 3.1 可用性损失

1. **设备故障** (Breakdown)
   - 设备损坏
   - 意外停机

2. **设置和调整** (Setup & Adjustment)
   - 换模换线
   - 启动调试

### 3.2 性能损失

3. **空转和短暂停机** (Idling & Minor Stoppages)
   - 微停机(<5分钟)
   - 传感器阻塞
   - 产品卡住

4. **速度降低** (Reduced Speed)
   - 设备磨损
   - 操作不当
   - 低于设计速度

### 3.3 质量损失

5. **过程不良** (Process Defects)
   - 不合格品
   - 需要返工

6. **启动损失** (Startup Rejects)
   - 试产不良
   - 预热期损失

## 4. 损失时间计算

本系统支持详细的损失时间分析:

```python
# 可用性损失(分钟)
availability_loss = 计划停机 + 非计划停机

# 性能损失(分钟)
performance_loss = 运行时间 - (理想周期 × 实际产量)

# 质量损失(分钟)
quality_loss = (不良品数 + 返工数) × 理想周期

# 总损失
total_loss = availability_loss + performance_loss + quality_loss
```

- 备件管理

### 5.2 提升性能率

- 工艺优化

### 5.3 提升合格率

- 持续改进

## 6. OEE数据采集要求

### 6.1 时间数据
- 班次开始/结束时间
- 停机开始/结束时间
- 停机原因分类

### 6.2 产量数据
- 计划产量
- 实际产量
- 合格数量
- 不良品数量
- 返工数量

### 6.3 设备数据
- 设备编号
- 理想周期时间
- 设备状态

### 6.4 数据采集方式
- **自动采集**: MES系统、传感器、PLC
- **半自动**: 扫码枪、触摸屏
- **手工录入**: 纸质记录、表单

## 8. 常见问题解答

### 8.1 OEE能否超过100%?

**答**: 严格来说不能。如果计算出超过100%,通常是以下原因:
- 理想周期时间设置过长
- 数据采集错误
- 性能率计算时应设置上限100%

### 8.2 不同行业的OEE标准一样吗?

**答**: 基本计算方法相同,但基准值可能不同:
- 离散制造(机加工): 60-70%
- 连续生产(化工): 80-90%
- 汽车制造: 75-85%

### 8.3 OEE低如何改进?

**答**: 
1. 找出最低的那个要素(可用率/性能率/质量率)
2. 分析该要素的主要损失
3. 制定针对性改进措施
4. PDCA循环持续改进

### 8.4 如何设置理想周期时间?

**答**:
- 参考设备设计参数
- 历史最佳记录
- 试运行测试
- 行业标准

## 9. OEE vs. 其他指标

| 指标 | 侧重点 | OEE的优势 |
|-----|--------|----------|
| 产量 | 数量 | 考虑质量和效率 |
| 良品率 | 质量 | 综合考虑时间和速度 |
| 设备利用率 | 时间 | 考虑质量和性能 |
| 生产效率 | 速度 | 全面综合指标 |

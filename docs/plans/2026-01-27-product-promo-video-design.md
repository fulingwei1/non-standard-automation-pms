# 测试设备产品宣传视频设计方案

## 项目概述

| 项目 | 内容 |
|------|------|
| 产品 | ICT/FCT/EOL 测试设备 |
| 用途 | 客户拜访演示 |
| 时长 | 90秒（1分30秒） |
| 风格 | 专业商务风（蓝白配色、简洁大气） |
| 重点 | 测试能力（测试项目、精度、速度） |
| 技术 | Remotion (React + TypeScript) |

## 视觉风格

- **主色调**: 深蓝 (#1e3a5f) + 白色 (#ffffff)
- **强调色**: 科技蓝 (#3b82f6)
- **字体**: 思源黑体（中文）+ Inter（英文/数字）
- **动画**: 流畅的淡入淡出 + 数据滚动效果

## 视频结构

| 时间段 | 场景 | 内容 | 时长 |
|--------|------|------|------|
| 0-5秒 | 开场 | 公司 Logo 动画入场 | 5秒 |
| 5-20秒 | 痛点引入 | 客户面临的测试挑战 | 15秒 |
| 20-60秒 | 核心能力 | 测试项目、精度、速度 | 40秒 |
| 60-75秒 | 技术亮点 | 4个差异化优势 | 15秒 |
| 75-90秒 | 结尾 | 联系方式 + CTA | 15秒 |

## 场景详细设计

### 场景 1：开场（0-5秒）

**视觉效果：**
- 深蓝色背景渐显
- 公司 Logo 从中心缩放入场，带轻微发光效果
- Logo 下方显示公司 Slogan

**动画：**
- Logo 缩放：0.8 → 1.0，配合 spring 弹性动画
- 文字淡入：延迟 0.5 秒后出现

### 场景 2：痛点引入（5-20秒）

**内容：** 展示客户面临的 3 个核心挑战

| 序号 | 痛点 | 图标 |
|------|------|------|
| 1 | 测试效率低，产能受限 | 时钟图标 |
| 2 | 漏测风险高，质量难保障 | 警告图标 |
| 3 | 人工成本高，依赖经验 | 工人图标 |

**动画：**
- 三个痛点依次从左侧滑入（间隔 3 秒）
- 每个痛点配合数字动画

### 场景 3：核心能力展示（20-60秒）

#### 3.1 测试项目展示（20-35秒）

| 测试类型 | 说明 |
|----------|------|
| ICT 在线测试 | 开短路、电阻、电容、二极管 |
| FCT 功能测试 | 电压、电流、通讯、逻辑功能 |
| EOL 终检测试 | 整机性能、老化筛选 |

#### 3.2 测试精度展示（35-45秒）

| 指标 | 数值 |
|------|------|
| 电压精度 | ±0.01% |
| 电流精度 | ±0.05% |
| 测量重复性 | 99.99% |

#### 3.3 测试速度展示（45-60秒）

| 指标 | 数值 | 对比 |
|------|------|------|
| 单板测试时间 | 3秒/片 | 传统 15秒 |
| 日产能 | 10,000+ 片 | 提升 5 倍 |
| 换线时间 | 5分钟 | 传统 30分钟 |

### 场景 4：技术亮点（60-75秒）

| 序号 | 亮点 | 说明 |
|------|------|------|
| 1 | 自研测试平台 | 软硬件一体化，稳定可靠 |
| 2 | 智能诊断 | AI 辅助故障定位 |
| 3 | 柔性兼容 | 一机多用，快速换型 |
| 4 | 数据追溯 | 全流程数据记录 |

### 场景 5：结尾（75-90秒）

- 公司 Logo + 名称
- Slogan：「精准测试，智造未来」
- 联系方式：电话 / 邮箱 / 官网
- CTA：「立即咨询，获取定制方案」

## 技术实现

- **框架**: Remotion 4.x
- **语言**: TypeScript + React
- **样式**: TailwindCSS
- **输出**: 1920x1080 MP4, 30fps

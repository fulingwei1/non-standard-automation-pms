# 生产主管操作手册

## 1. 角色职责

生产主管是生产进度模块的核心用户，主要负责：
- ✅ 审核和调整生产排程
- ✅ 监控生产进度和产能利用率
- ✅ 处理生产异常和瓶颈
- ✅ 审核报工记录
- ✅ 质量监控和改进

## 3. 生产排程管理

### 3.1 查看排程计划

**路径**: 生产管理 > 排程计划

**操作步骤**:
1. 进入【排程计划】页面
2. 选择车间和日期范围
3. 查看排程甘特图和列表
4. 可按工单号、项目名搜索

**关键指标**:
- 🟢 **完工率**: 已完成工单数 / 总工单数
- 🟡 **准时率**: 按时完成 / 已完成工单
- 🔴 **延期数**: 超期未完成的工单数

### 3.2 AI智能排程

**使用场景**: 多订单同时到达，需要优化排程方案

**操作步骤**:
1. 点击【AI智能排程】按钮
2. 选择待排程的订单（可多选）
3. 设置约束条件:
   - 最大加班时间
   - 是否优先技能匹配
   - 交期是否严格
4. 选择优化算法（推荐：遗传算法）
5. 点击【开始优化】，等待计算完成
6. 查看优化结果对比:
   - 优化前总完工时间 vs 优化后
   - 资源利用率对比
7. 确认应用排程方案

**注意事项**:
- ⚠️ AI排程可能需要1-2分钟计算时间
- ⚠️ 确认应用后将覆盖原排程
- ⚠️ 建议先预览再确认

### 3.3 手动调整排程

**操作步骤**:
1. 在甘特图中拖拽工单条调整时间
2. 或点击工单，在弹窗中修改:
   - 计划开始时间
   - 计划结束时间
   - 指派工人
   - 指派设备
3. 点击【保存】

**系统自动检查**:
- ✅ 资源冲突检测
- ✅ 技能匹配检验
- ✅ 工时合理性校验

## 5. 报工审核

### 5.1 待审核列表

**路径**: 生产管理 > 报工审核

**列表显示**:
- 工单号
- 工人姓名
- 报工时间
- 完成数量
- 工时
- 异常标识 (如有)

**筛选条件**:
- 状态: 待审核/已通过/已拒绝
- 日期范围
- 工人
- 工单

### 5.2 审核报工

**审核要点**:
1. **数量合理性**:
   - 完成数量 ≤ 工单计划数量 - 已完成数量
   - 合格数 + 不良数 = 完成数量

2. **工时合理性**:
   - 单位工时 = 工时 / 完成数量
   - 对比标准工时，偏差不超过±30%
   - 单日工时不超过12小时

3. **异常标记**:
   - 🔴 单日工时超12小时
   - 🟡 工时偏差超30%
   - 🟠 连续7天无休息

**操作步骤**:
1. 点击报工记录查看详情
2. 查看工作描述和照片(如有)
3. 确认数据准确性
4. 选择【通过】或【拒绝】
5. 填写审核意见
6. 如需调整，输入调整后的工时/数量
7. 点击【提交】

**批量审核**:
- 勾选多条记录
- 点击【批量通过】
- 确认操作

## 6. 质量监控

### 6.1 质量看板

**路径**: 质量管理 > 质量看板

**关键指标**:
```
今日质量数据
├─ 检验批次: 32
├─ 检验数量: 1,600
├─ 合格数量: 1,552
├─ 不良数量: 48
└─ 不良率: 3.0% 🟡 (阈值5%)
```

**不良类型分布**:
- 焊接不良: 37.5%
- 尺寸偏差: 31.25%
- 外观不良: 31.25%

### 6.2 SPC控制图

**查看步骤**:
1. 选择工序或物料
2. 选择测量类型（如：焊点直径）
3. 设置时间范围
4. 点击【生成SPC图】

**图表解读**:
```
焊点直径 SPC控制图
       UCL 5.61 ─────────────────────────
              ·     ·  ·
       μ  5.25 ─·──·─·──·─·──·─·──·─·─
              ·  ·        ·        ·
       LCL 4.89 ─────────────────────────
```

**失控判断**:
- 🔴 点超出UCL/LCL → 立即查明原因
- 🟡 连续7点在均值同侧 → 趋势异常预警
- 🟠 3点中2点在2σ外 → 可能失控

**处理措施**:
1. 点击失控点查看详情
2. 通知质检员/工艺工程师
3. 暂停生产并排查原因
4. 采取纠正措施
5. 验证改善效果

### 6.3 不良分析处理

**查看不良分析**:
1. 进入【不良分析】页面
2. 查看待处理的分析单
3. 点击单号查看根因分析（5M1E）:
   - 人因 (Man)
   - 机因 (Machine)
   - 料因 (Material)
   - 法因 (Method)
   - 测因 (Measurement)
   - 环因 (Environment)

**跟踪纠正措施**:
- 查看纠正措施内容
- 确认责任人
- 跟踪完成期限
- 验证改善效果

## 8. 常见问题处理

### 8.1 工单延期

**问题**: 工单预计延期，如何处理？

**解决步骤**:
1. 查看进度偏差原因
2. 评估延期影响（后续工序/交期）
3. 采取措施:
   - **方案A**: 增加人力（优先）
   - **方案B**: 延长工作时间
   - **方案C**: 调整后续工单顺序
   - **方案D**: 与客户协商延期
4. 更新排程计划
5. 通知相关人员

### 8.2 质量异常

**问题**: 不良率突然升高

**处理流程**:
1. 查看SPC控制图确认异常
2. 通知质检员现场确认
3. 暂停相关生产
4. 组织分析会议（生产/质量/工艺）
5. 制定纠正措施
6. 验证改善效果
7. 恢复生产

### 8.3 设备故障

**问题**: 关键设备故障导致停产

**应急措施**:
1. 报修设备（维修团队）
2. 评估影响范围
3. 调整排程:
   - 将工单转移到备用设备
   - 调整其他工单顺序
   - 通知相关工人
4. 更新预计完工时间
5. 跟踪维修进度

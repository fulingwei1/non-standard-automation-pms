## 3. 质量检验操作

### 3.1 接收检验任务

**路径**: 质检管理 > 待检任务

**操作**:
1. 查看待检任务列表
2. 点击【开始检验】
3. 系统自动分配检验单号

### 3.2 过程检验 (IPQC)

**步骤**:

#### Step 1: 扫描工单条码
- 使用扫码枪扫描工单条码
- 或手动输入工单号

#### Step 2: 填写检验信息
```
检验数量: [50] 件
抽样方案: GB/T 2828.1 (自动计算)
抽样数量: [13] 件  ← 系统自动计算
```

#### Step 3: 记录测量数据

**尺寸测量示例**:
| 序号 | 测量项 | 规格 | 实测值 | 判定 |
|------|--------|------|--------|------|
| 1 | 焊点直径 | 5.0±0.5mm | 5.23 | ✅ 合格 |
| 2 | 焊点直径 | 5.0±0.5mm | 5.68 | ❌ 超上限 |

**系统自动判定**:
- 绿色✅: 在规格范围内
- 红色❌: 超出规格

#### Step 4: 不良记录

**发现不良时**:
1. 点击【记录不良】
2. 选择不良类型:
   - 焊接不良
   - 尺寸偏差
   - 外观不良
   - 功能异常
   - 其他
3. 输入不良描述
4. 拍照上传 (可选)
5. 填写不良数量

#### Step 5: 综合判定

**判定规则**:
```
不良率 = 不良数量 / 检验数量
- 不良率 = 0%      → 批量合格
- 不良率 < 5%      → 部分合格 (需返工)
- 不良率 ≥ 5%      → 批量不合格 (全检)
```

**选择处理方式**:
- 🟢 **合格**: 放行入库
- 🟡 **返工**: 创建返工单
- 🔴 **让步接收**: 需主管审批
- ⚫ **报废**: 记录报废原因

#### Step 6: 提交检验结果
- 点击【提交】
- 系统自动更新工单状态
- 打印检验报告 (可选)

## 4. 不良品处理

### 4.1 创建返工单

**触发条件**: 检验结果为"返工"

**自动创建**:
1. 系统自动生成返工单号
2. 关联原工单和质检记录
3. 填写返工数量
4. 分配返工工人
5. 提交

### 4.2 让步接收申请

**使用场景**: 轻微不良，不影响功能，申请让步接收

**操作**:
1. 点击【申请让步接收】
2. 填写申请理由
3. 上传照片证据
4. 提交审批
5. 等待主管批准

**审批流程**:
```
质检员申请 → 质量主管审批 → 项目经理确认 → 客户同意(重大问题)
```

### 4.3 报废处理

**触发条件**: 无法返工的严重不良

**操作**:
1. 填写报废原因
2. 拍照记录
3. 提交审批
4. 等待财务确认
5. 打印报废标签

## 5. SPC数据分析

### 5.1 查看控制图

**路径**: 质检管理 > SPC分析

**操作**:
1. 选择工序/产品
2. 选择测量项目
3. 设置时间范围
4. 点击【生成控制图】

**控制图示例**:
```
   UCL ─────────────────
       ·  ·    ·    ·
   μ   ─·──·──·──·──·──
            ·       ·
   LCL ─────────────────
```

### 5.2 异常点处理

**发现异常点**:
- 🔴 超出控制限 → 立即通知主管
- 🟡 连续趋势 → 记录预警

**操作**:
1. 点击异常点查看详情
2. 分析可能原因:
   - 设备温度漂移
   - 原材料批次变更
   - 操作工换人
3. 填写处理措施
4. 跟踪改善效果

## 10. 常见问题

**Q1: 测量值在边界，如何判定？**
A: 下限含等号，上限不含。如规格5.0±0.5mm，则4.5合格，5.5不合格。

**Q2: 发现批量不良如何处理？**
A: 立即通知主管，暂停放行，全检或返工。

**Q3: 让步接收被拒绝怎么办？**
A: 按正常流程返工或报废，不可私自放行。

**Q4: 检验设备故障如何处理？**
A: 立即停止使用，通知计量室维修，使用备用设备。

## 4. 报工操作

### 4.1 开工报工
**第一次开始工作时**:
1. 点击【开工】按钮
2. 确认工单信息
3. 扫描物料条码 (可选)
4. 点击【开始工作】

**系统记录**:
- 开工时间
- 工位信息
- 设备信息

### 4.2 进度报工
**工作过程中定期汇报**:

**步骤**:
1. 点击【报工】按钮
2. 填写报工信息:
```
工作时间: [2.5] 小时 (自动计算距上次报工的时间)
完成数量: [20] 件
合格数量: [19] 件
不良数量: [1] 件
```
3. 如有不良品:
   - 选择不良类型
   - 拍照上传
   - 填写描述
4. 填写工作描述 (可选)
5. 点击【提交】

**注意**:
- ⚠️ 每完成一定数量及时报工
- ⚠️ 不要等到下班才报工
- ⚠️ 数量要准确，系统会校验

### 4.3 暂停报工
**需要暂停工作时**:
1. 点击【暂停】按钮
2. 选择暂停原因:
   - 等待物料
   - 设备故障
   - 休息时间
   - 其他
3. 填写暂停说明
4. 点击【确认暂停】

**恢复工作**:
- 点击【恢复工作】
- 系统自动记录暂停时长

### 4.4 完工报工
**全部完成时**:
1. 点击【完工】按钮
2. 确认最终数据:
```
计划数量: 200件
实际完成: 200件
合格数量: 196件
不良数量: 4件
总工时: 16.5小时
```
3. 填写完工说明
4. 点击【提交完工】
5. 打印完工标签 (可选)

## 5. 移动端报工 (推荐)

### 5.1 扫码报工
**最快捷的方式**:
1. 打开手机APP
2. 点击【扫码报工】
3. 扫描工单条码
4. 自动显示工单信息
5. 输入完成数量
6. 拍照记录 (如有不良)
7. 点击【提交】

### 5.2 语音报工
**解放双手**:
1. 点击【语音报工】图标
2. 说出报工信息:
   "工单001，完成20件，1件不良"
3. 系统自动识别并填写
4. 确认信息
5. 提交

### 5.3 离线报工
**无网络时**:
- APP自动保存到本地
- 恢复网络后自动上传
- 提示同步成功

## 6. 异常处理

### 6.1 物料短缺
**发现物料不足时**:
1. 点击【异常上报】
2. 选择【物料短缺】
3. 填写缺料信息
4. 拍照上传
5. 提交 (自动通知物料员)
6. 暂停工单等待处理

### 6.2 设备故障
**设备出现问题**:
1. 立即停止工作
2. 点击【异常上报】
3. 选择【设备故障】
4. 描述故障现象
5. 拍照/录视频
6. 提交 (自动通知维修)

### 6.3 质量问题
**发现批量不良**:
1. 停止生产
2. 隔离不良品
3. 上报异常
4. 拍照记录
5. 通知质检员
6. 等待处理指示

## 10. 常见问题

**Q1: 忘记报工怎么办？**
A: 可补报，但需注明补报原因，需主管审批。

**Q2: 报工数据填错了如何修改？**
A: 提交后无法修改，联系主管撤回后重新提交。

**Q3: 手机没电了怎么报工？**
A: 可使用车间公用平板或下班后补报。

**Q4: 不良品如何处理？**
A: 隔离存放，贴上不良标签，等待质检处理。

**Q5: 标准工时完不成怎么办？**
A: 如实报工，说明原因，不要少报数量或多报工时。

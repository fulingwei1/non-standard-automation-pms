# 异常处理手册

## 1. 概述

本手册提供生产过程中各类异常的处理流程和应急措施。

**异常分类**:
- 🔴 **A类**: 严重异常（停产/安全事故）
- 🟡 **B类**: 重要异常（影响交期/质量）
- 🟢 **C类**: 一般异常（可快速解决）

---

## 2. 设备异常

### 2.1 设备故障

**症状**:
- 设备无法启动
- 运行中突然停机
- 异常噪音/震动
- 报警灯亮起

**应急处理** (5分钟内):
1. ⚠️ **立即停机**,按下急停按钮
2. 📋 **隔离现场**,拉警戒线
3. 📞 **报告主管**,说明故障现象
4. 📸 **拍照记录**,保留现场
5. 🔒 **挂故障牌**,禁止他人操作

**系统操作**:
```
异常上报流程:
1. 打开APP → 异常上报
2. 选择设备故障
3. 设备名称: [自动焊接机-01]
4. 故障现象: [突然停机，显示屏报警]
5. 上传照片
6. 提交 → 自动通知维修组
```

**等级判定**:
- 🔴 **A级**: 关键设备，影响整条产线 → 2小时内必须修复
- 🟡 **B级**: 一般设备，有备用 → 4小时内修复
- 🟢 **C级**: 辅助设备 → 当天修复

**处理流程**:
```
工人上报 → 主管确认 → 维修组抢修 → 试运行 → 验收 → 恢复生产
            ↓
         调整排程(如影响交期)
```

**预防措施**:
- 每日点检（开机前5分钟检查）
- 定期保养（按保养计划执行）
- 及时润滑（听到异响立即处理）

---

### 2.2 设备参数漂移

**症状**:
- 产品尺寸/质量不稳定
- SPC控制图显示趋势异常
- 频繁微调仍无法达标

**处理步骤**:
1. 停止生产
2. 隔离已生产品（待检）
3. 通知工艺工程师
4. 校准设备参数
5. 首件确认合格后恢复生产

**案例**:
```
案例: 焊接温度漂移
现象: 焊点虚焊率从1%上升至8%
原因: 温控模块老化
处理: 更换温控器，重新校准温度
时长: 1.5小时
影响: 暂停生产，隔离80件产品全检
```

## 3. 质量异常

### 3.1 批量不良

**触发条件**:
- 不良率 > 5%
- 连续5件不良
- 严重功能缺陷

**紧急处理** (立即):
1. ⛔ **停止生产**
2. 🔒 **隔离不良品**,贴红色标签
3. 📋 **隔离在制品**,等待检验
4. 📞 **通知质检员**和主管
5. 🚫 **禁止使用**该批物料/设备

**5WHY分析法**:
```
案例: 焊接不良率突增至12%

为什么不良率高?
→ 焊点虚焊

为什么虚焊?
→ 焊接温度不够

为什么温度不够?
→ 温控器故障

为什么温控器故障?
→ 超期使用未更换

为什么未更换?
→ 保养计划遗漏

根本原因: 保养计划不完善
纠正措施: 
1. 更换温控器
2. 完善保养计划
3. 加强点检培训
```

**处理流程**:
```
发现批量不良
    ↓
停止生产 + 隔离
    ↓
通知质检员 + 主管
    ↓
不良分析(5M1E)
    ↓
制定纠正措施
    ↓
实施改善
    ↓
验证效果(首件确认)
    ↓
恢复生产
```

**返工处理**:
```
系统自动创建返工单:
├─ 返工单号: RW-2026021701
├─ 原工单: WO-001
├─ 返工数量: 48件
├─ 返工原因: 焊接虚焊
├─ 指派工人: 张三(技能等级4)
├─ 预计工时: 4小时
└─ 要求完成: 今日17:00前
```

### 3.2 客户投诉

**接到投诉时**:
1. 记录投诉内容和证据
2. 立即上报质量经理
3. 48小时内回复初步原因
4. 7天内提交8D报告

**8D报告模板**:
```
D1-团队组建: 质量/生产/工艺/供应商
D2-问题描述: 焊点强度不足,客户测试失效
D3-临时措施: 召回该批次产品,免费返工
D4-根本原因: 焊料成分不合格
D5-纠正措施: 更换合格焊料,加强来料检验
D6-验证措施: 全检+抽样破坏性试验
D7-预防措施: 供应商审核+首件确认
D8-团队表彰: 感谢参与人员
```

## 4. 物料异常

### 4.1 物料短缺

**发现缺料**:
1. 立即上报系统
2. 通知物料员
3. 暂停工单
4. 切换到其他工单

**系统操作**:
```
异常上报 > 物料短缺
├─ 工单号: WO-003
├─ 缺料名称: 螺丝M5×20
├─ 需求数量: 500个
├─ 剩余库存: 0
├─ 紧急程度: 高(影响交期)
└─ 提交 → 自动通知物料员
```

**物料员处理**:
- 优先调配其他车间库存
- 紧急采购（当天到货）
- 通知工人预计到料时间

**临时措施**:
- 先做其他工序
- 转产其他工单
- 申请加班补进度

---

### 4.2 物料质量问题

**发现来料不良**:
1. 停止使用
2. 隔离该批物料
3. 贴"待检"标签
4. 通知质检员IQC检验
5. 拍照记录

**系统记录**:
```
来料异常报告:
├─ 物料编码: MT-12345
├─ 批次号: 20260215-A
├─ 供应商: XX五金厂
├─ 异常类型: 尺寸偏差
├─ 不良数量: 50/100
├─ 不良率: 50%
├─ 处理建议: 退货
└─ 照片: [上传3张]
```

**处理方式**:
- 退货: 严重不良,无法使用
- 让步接收: 轻微偏差,不影响功能
- 挑选使用: 部分合格可用
- 返工加工: 可返工至合格

## 5. 人员异常

### 5.1 人员短缺

**原因**:
- 请假/离职
- 培训/会议
- 紧急任务

**应急方案**:
```
人力调配决策树:
缺1人:
  → 延长工时(加班2小时)
  或
  → 调配其他车间人员(临时支援)

缺2-3人:
  → 调整排程,推迟非紧急订单
  或  
  → 申请临时工

缺>3人:
  → 上报生产经理,启动应急预案
```

**系统支持**:
```
查看可调配人员:
├─ 同技能工人: 3人
│   ├─ 李四(车间B) 负荷70%  ← 可调配
│   ├─ 王五(车间A) 负荷95%
│   └─ 赵六(车间C) 负荷60%  ← 可调配
└─ 建议: 调配李四支援2小时
```

---

### 5.2 工伤事故

**发生工伤** (立即):
1. 🚨 **停止作业**,切断电源
2. 🏥 **急救处理**,拨打120
3. 📞 **上报安全员**和主管
4. 🔒 **保护现场**,拍照记录
5. ⚠️ **疏散人员**,确保安全

**严重程度**:
- 🔴 **重伤**: 骨折/大出血/休克 → 立即送医
- 🟡 **轻伤**: 擦伤/轻微烫伤 → 医务室处理
- 🟢 **未受伤**: 惊吓 → 安抚休息

**事故调查**:
- 24小时内成立调查组
- 7天内提交事故报告
- 制定预防措施
- 全员安全培训

**系统记录**:
```
工伤事故报告:
├─ 事故时间: 2026-02-17 14:30
├─ 事故地点: 车间A 组装区
├─ 受伤人员: 张三
├─ 受伤部位: 右手食指
├─ 伤情: 切割伤,缝合3针
├─ 事故原因: 未佩戴防护手套
├─ 纠正措施: 
│   1. 强制佩戴防护用品
│   2. 增加安全培训
│   3. 每日安全提醒
└─ 责任人: 主管连带责任
```

## 6. 系统异常

### 6.1 系统故障

**无法登录**:
1. 检查网络连接
2. 清除浏览器缓存
3. 更换浏览器尝试
4. 联系IT支持: 8888

**数据丢失**:
1. 立即停止操作
2. 联系IT支持
3. 说明丢失数据内容
4. 等待恢复（24小时内）

**系统卡顿**:
- 关闭多余标签页
- 重启浏览器
- 清理缓存
- 升级浏览器

### 6.2 网络中断

**断网应急**:
1. 切换到离线模式(APP自动切换)
2. 数据保存在本地
3. 网络恢复后自动同步
4. 检查数据完整性

**长时间断网** (>2小时):
- 纸质记录报工数据
- 网络恢复后补录系统
- 注明"补录"标识

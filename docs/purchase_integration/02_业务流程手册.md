## 1. 采购管理流程

### 1.1 常规采购流程

**流程图**:
```
创建采购申请 → 部门主管审批 → 采购员询价/比价 → 创建采购订单 
→ 采购主管审批 → 发送供应商 → 供应商确认 → 跟踪订单 
→ 收货入库 → 质量检验 → 财务对账 → 完成
```

**详细步骤**:

#### 步骤1: 创建采购申请

**操作人**: 需求部门/PMC  
**路径**: 采购管理 → 采购申请 → 新建

**操作说明**:
1. 填写基本信息
   - 申请类型: 常规/紧急/缺料
   - 项目名称 (如果关联项目)
   - 申请原因
   
2. 添加采购物料明细
   - 选择物料 (支持搜索)
   - 输入需求数量
   - 选择需求日期
   - 填写用途说明
   - 设置紧急程度

3. 提交审批
   - 检查明细完整性
   - 点击"提交审批"
   - 系统自动发送审批通知

**注意事项**:
- 需求数量需考虑最小订购量
- 需求日期需考虑采购周期
- 紧急采购需注明原因

#### 步骤2: 采购申请审批

**操作人**: 部门主管/采购主管  
**路径**: 审批中心 → 待审批

**审批要点**:
- ✅ 物料需求合理性
- ✅ 数量准确性
- ✅ 日期可行性
- ✅ 预算充足性

**操作**:
- 批准: 点击"同意"，流转到采购员
- 驳回: 点击"驳回"，填写驳回原因

#### 步骤3: 询价比价 (可选)

**操作人**: 采购员  
**路径**: 采购管理 → 询价管理 → 新建询价单

**操作说明**:
1. 选择已审批的采购申请
2. 选择候选供应商 (3-5家)
3. 发送询价单
4. 收集供应商报价
5. 录入报价信息
6. 比价分析，选择最优供应商

**比价考虑因素**:
- 单价和总价
- 交货周期
- 付款条件
- 质量保证
- 供应商绩效评分

#### 步骤4: 创建采购订单

**操作人**: 采购员  
**路径**: 采购管理 → 采购订单 → 新建

**操作说明**:
1. 填写订单基本信息
   - 选择供应商
   - 订单类型 (常规/紧急)
   - 关联采购申请
   
2. 添加订单明细
   - 从采购申请导入
   - 确认数量和单价
   - 确认交货日期
   - 输入收货地址

3. 设置付款条件
   - 选择付款方式
   - 设置付款节点

4. 保存并提交审批

**注意事项**:
- 单价需与报价单一致
- 交货日期需供应商确认
- 总金额自动计算

#### 步骤5: 采购订单审批

**操作人**: 采购主管/财务主管  
**路径**: 审批中心 → 待审批

**审批权限**:
- < 5万元: 采购主管审批
- 5-20万元: 采购主管 + 财务主管
- > 20万元: 采购主管 + 财务主管 + 总经理

#### 步骤6: 发送供应商

**操作人**: 采购员  
**操作**: 订单审批通过后，点击"发送供应商"

**系统自动**:
- 生成PDF格式订单
- 发送邮件给供应商
- 记录发送时间

#### 步骤7: 订单跟踪

**操作人**: 采购员  
**路径**: 采购管理 → 订单跟踪

**跟踪内容**:
- 供应商确认状态
- 承诺交货日期
- 生产进度
- 预计到货时间
- 延期风险

**操作**:
- 定期联系供应商
- 更新订单状态
- 延期订单标红提醒

#### 步骤8: 收货入库

**操作人**: 仓管员  
**路径**: 库存管理 → 收货管理 → 新建收货单

**操作说明**:
1. 扫描/选择采购订单
2. 核对物料和数量
3. 录入实收数量
4. 选择入库仓位
5. 生成批次号
6. 打印入库标签
7. 提交入库

**注意事项**:
- 实收数量可能与订单数量不同
- 需记录供应商送货单号
- 异常情况需拍照记录

#### 步骤9: 质量检验

**操作人**: 质检员  
**路径**: 质量管理 → 来料检验

**检验内容**:
- 外观检查
- 规格核对
- 功能测试
- 数量复核

**检验结果**:
- ✅ 合格: 转正式库存
- ❌ 不合格: 隔离并通知采购退货
- ⚠️ 部分合格: 分别处理

### 1.2 紧急采购流程

**特点**: 流程简化，审批加速

```
缺料预警 → 自动生成采购建议 → 紧急采购申请 → 快速审批 
→ 紧急采购订单 → 直接发送供应商 → 加急配送 → 收货入库
```

**审批权限**:
- 关键物料缺料: 自动审批
- 非关键物料: 采购主管一人审批

**供应商选择**:
- 优先选择绩效高、交期短的供应商
- 可接受价格溢价

## 2. 库存管理流程

### 2.1 入库流程

```
收货 → 质检 → 入库 → 生成批次 → 打印标签 → 上架 → 更新库存
```

**详细操作**: 见采购流程 步骤8-9

### 2.2 领料流程

#### 步骤1: 物料预留 (可选)

**操作人**: PMC/生产计划员  
**路径**: 库存管理 → 物料预留 → 新建预留单

**操作说明**:
1. 选择项目/工单
2. 添加需要预留的物料
3. 输入预留数量
4. 设置预计使用日期
5. 提交预留

**系统自动**:
- 检查可用库存
- 扣减可用库存
- 增加预留库存
- 生成预留单号

#### 步骤2: 创建领料单

**操作人**: 生产部门/仓管员  
**路径**: 库存管理 → 领料管理 → 新建领料单

**操作说明**:
1. 选择工单/项目
2. 添加领料物料
   - 自动带出预留物料
   - 可手动添加其他物料
3. 输入领用数量
4. 选择领料仓位
5. 提交审批 (如需要)

#### 步骤3: 出库操作

**操作人**: 仓管员  
**操作**:
1. 审核领料单
2. 拣货
3. 扫描批次号
4. 确认出库
5. 打印领料单

**系统自动**:
- 扣减库存数量
- 释放预留
- 创建交易记录
- 更新成本

### 2.3 退料流程

**操作人**: 生产部门  
**路径**: 库存管理 → 退料管理 → 新建退料单

**适用场景**:
- 多领物料退回
- 物料质量问题
- 项目取消

**操作**:
1. 选择原领料单
2. 选择退料物料
3. 输入退料数量
4. 填写退料原因
5. 提交退料

**仓管员操作**:
1. 审核退料单
2. 检查物料状态
3. 确认入库
4. 更新库存

## 3. 缺料预警处理流程

### 3.1 预警生成

**触发方式**:
- 定时扫描 (每天 08:00, 14:00, 18:00)
- 手动触发扫描
- 库存变动实时检测

**预警条件**:
1. 当前库存 < 安全库存
2. 项目需求 > 可用库存
3. 预测未来缺料

### 3.2 预警处理流程

```
预警生成 → 影响分析 → 推荐处理方案 → 选择方案 → 执行 → 跟踪 → 解决
```

#### 步骤1: 预警查看

**操作人**: PMC/采购员  
**路径**: 缺料预警中心 → 预警列表

**预警信息**:
- 物料名称
- 缺料数量
- 预警级别 (低/中/高/紧急)
- 影响项目
- 预计影响天数
- 需求日期

#### 步骤2: 影响分析

**操作**: 点击预警详情

**分析内容**:
- 受影响的工单/项目
- 是否在关键路径上
- 预计延期天数
- 成本影响
- 客户影响

#### 步骤3: 查看处理方案

**系统自动推荐**:

**方案1: 紧急采购**
- 优点: 快速补料
- 缺点: 成本较高
- 预计交期: 3-5天
- 推荐供应商: XXX (绩效A级)

**方案2: 使用替代料**
- 优点: 无需等待
- 缺点: 需技术确认
- 可用库存: 50件
- 价格差异: +15%

**方案3: 重新排期**
- 优点: 无额外成本
- 缺点: 项目延期
- 预计延期: 7天
- 需客户确认

**方案4: 分批生产**
- 优点: 部分交付
- 缺点: 增加生产成本
- 首批可完成: 60%

#### 步骤4: 选择并执行方案

**操作人**: PMC/采购主管  
**操作**:
1. 评估各方案
2. 选择最优方案
3. 点击"执行方案"

**系统自动**:
- 创建采购申请 (如选紧急采购)
- 创建变更单 (如选重新排期)
- 创建替换申请 (如选替代料)

#### 步骤5: 跟踪处理进度

**路径**: 缺料预警中心 → 处理中

**跟踪内容**:
- 方案执行状态
- 采购订单进度
- 预计到货时间
- 风险变化

#### 步骤6: 预警解决

**条件**:
- 物料已到货
- 库存已补足
- 风险已消除

**操作**: 点击"标记为已解决"

## 4. 供应商管理流程

### 4.1 供应商入驻

**操作人**: 采购员  
**路径**: 供应商管理 → 新增供应商

**所需信息**:
- 基本信息: 名称、代码、联系人
- 资质信息: 营业执照、质量认证
- 财务信息: 开户行、账号、税号
- 供货范围: 可供应物料类别
- 合作条款: 付款方式、交货周期

### 4.2 供应商绩效评估

**评估周期**: 每季度  
**评估人**: 采购主管  
**路径**: 供应商管理 → 绩效评估

**评估维度**:

| 维度 | 权重 | 数据来源 |
|------|------|----------|
| 准时交货率 | 40% | 采购订单交货记录 |
| 质量合格率 | 30% | 来料检验记录 |
| 价格竞争力 | 20% | 采购价格 vs 市场价格 |
| 响应速度 | 10% | 报价响应时间 |

**评分标准**:
- A级 (90-100分): 优秀供应商，优先合作
- B级 (75-89分): 合格供应商，正常合作
- C级 (60-74分): 一般供应商，观察改进
- D级 (<60分): 不合格，考虑淘汰

**系统自动计算**:
```
综合评分 = 准时交货率×40% + 质量合格率×30% + 
          价格竞争力×20% + 响应速度×10%
```

### 4.3 供应商淘汰

**触发条件**:
- 连续2个季度评分 < 60分
- 严重质量事故
- 严重延期导致停产
- 商业欺诈行为

**流程**:
1. 发出整改通知
2. 给予30天整改期
3. 复评仍不合格
4. 启动淘汰流程
5. 寻找替代供应商
6. 完成切换后冻结

## 5. 盘点管理流程

### 5.1 定期盘点

**盘点周期**:
- 原材料: 每月盘点
- 半成品: 每月盘点
- 成品: 每周盘点
- 关键物料: 每周盘点

**流程**:
```
创建盘点任务 → 冻结库存 → 盘点执行 → 录入实盘 → 差异分析 
→ 调整审批 → 库存调整 → 解冻库存 → 生成报告
```

#### 步骤1: 创建盘点任务

**操作人**: 仓库主管  
**路径**: 库存管理 → 盘点管理 → 新建盘点任务

**操作**:
1. 选择盘点类型 (全盘/抽盘/循环盘)
2. 选择盘点范围 (仓库/库区/物料类别)
3. 设置盘点日期
4. 分配盘点人员
5. 生成盘点单

#### 步骤2: 冻结库存

**目的**: 盘点期间禁止出入库，保证数据一致性

**系统自动**:
- 标记库存为"盘点中"
- 暂停出入库操作
- 记录账面数量

#### 步骤3: 盘点执行

**操作人**: 仓管员  
**工具**: 盘点单 + PDA/手持终端

**操作**:
1. 按盘点单逐项清点
2. 扫描物料条码
3. 录入实际数量
4. 记录异常情况
5. 复盘 (差异大于5%需复盘)

#### 步骤4: 差异分析

**操作人**: 仓库主管  
**路径**: 盘点任务 → 差异分析

**分析内容**:
- 盘盈盘亏物料
- 差异数量和金额
- 差异原因分析
- 责任人确认

**常见差异原因**:
- 未及时录入单据
- 领料/退料未办手续
- 物料摆放错位
- 计数错误
- 物料损耗

#### 步骤5: 调整审批

**审批权限**:
- 差异金额 < 1000元: 仓库主管
- 1000-5000元: 仓库主管 + 财务
- > 5000元: 仓库主管 + 财务 + 总经理

#### 步骤6: 库存调整

**操作**: 审批通过后，点击"执行调整"

**系统自动**:
- 更新库存数量
- 创建调整单
- 记录账务处理
- 生成差异报告

#### 步骤7: 解冻库存

**操作**: 盘点完成后，点击"解冻库存"

**系统自动**:
- 恢复正常出入库
- 更新库存状态
- 发布盘点报告

## 6. 常见问题处理

### Q1: 采购订单延期怎么办？

**处理步骤**:
1. 联系供应商确认原因和新的交货日期
2. 评估对生产的影响
3. 通知PMC和相关项目经理
4. 考虑以下措施:
   - 寻找替代供应商紧急采购
   - 调整生产计划
   - 使用替代料
   - 通知客户延期
5. 记录延期原因，纳入供应商绩效考核

### Q2: 库存数据不准确怎么办？

**排查步骤**:
1. 检查近期出入库单据是否都已录入
2. 查看是否有未审核的单据
3. 检查盘点调整是否已执行
4. 查询交易记录，找出差异点
5. 必要时进行紧急盘点

### Q3: 缺料预警误报怎么办？

**常见原因**:
1. 在途物料未及时更新状态
2. 预留物料未释放
3. 安全库存设置不合理
4. 项目需求数据错误

**解决方法**:
1. 更新采购订单状态
2. 清理无效预留
3. 调整安全库存参数
4. 修正项目BOM数据

### Q4: 如何加快采购审批速度？

**方法**:
1. 启用自动审批规则 (金额阈值内)
2. 使用移动端审批
3. 设置代理审批人
4. 批量审批功能
5. 紧急采购走快速通道

✅ **精益库存**: 降低库存周转天数  
✅ **ABC分类**: 重点管理A类物料  
✅ **先进先出**: 严格执行FIFO原则  
✅ **定期盘点**: 关键物料每周盘点  
✅ **5S管理**: 保持仓库整洁有序

✅ **及时响应**: 预警24小时内处理  
✅ **根因分析**: 找出缺料根本原因  
✅ **预防为主**: 优化安全库存设置  
✅ **协同沟通**: PMC、采购、生产联动  
✅ **数据分析**: 定期分析缺料趋势

# 项目报告文档

本目录包含项目的各种分析报告、重构报告和优化报告。

## 报告分类

### 代码质量报告
- `CODE_QUALITY_*.md` - 代码质量分析报告
- `code_quality_analysis_report.json` - 代码质量分析数据

### 重构报告
- `API_REFACTORING_GUIDE.md` - API重构指南
- `BACKEND_REFACTORING_GUIDE.md` - 后端重构指南
- `COMPONENT_REFACTORING_*.md` - 组件重构相关报告
- `REFACTORING_*.md` - 重构完成报告

### 代码拆分报告
- `API_SPLITTING_*.md` - API拆分报告
- `CODE_SPLITTING_*.md` - 代码拆分报告
- `BACKEND_API_SPLIT_*.md` - 后端API拆分报告

### 性能优化报告
- `PERFORMANCE_OPTIMIZATION_*.md` - 性能优化报告
- `performance_optimization_report_*.json` - 性能优化数据

### 测试覆盖率报告
- `TEST_COVERAGE_*.md` - 测试覆盖率报告
- `coverage_*.json` - 覆盖率数据

### 结构分析报告
- `CODE_STRUCTURE_ANALYSIS.md` - 代码结构分析
- `LARGE_COMPONENTS_ANALYSIS.md` - 大组件分析
- `FINAL_CODEBASE_ANALYSIS.md` - 最终代码库分析

## 说明

这些报告记录了项目开发过程中的各种分析和优化工作，保留用于参考和历史记录。

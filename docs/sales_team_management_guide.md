# 销售团队管理使用手册

## 一、功能概述

销售团队管理模块提供了完整的销售组织架构、目标管理和区域管理功能，帮助企业：

- 建立清晰的销售团队组织架构
- 设定和分解销售目标
- 管理销售区域划分
- 跟踪目标达成情况
- 进行团队和个人业绩排名

## 二、快速开始

### 2.1 创建销售团队

1. 进入「销售管理」→「团队管理」
2. 点击「新建团队」
3. 填写团队信息：
   - 团队编码：唯一标识（如 T001）
   - 团队名称：团队名称（如华东团队）
   - 团队类型：选择区域团队/产品团队/行业团队
   - 上级团队：选择上级团队（可选）
   - 团队负责人：选择负责人
4. 点击「保存」

### 2.2 添加团队成员

1. 进入团队详情页
2. 点击「成员管理」→「添加成员」
3. 选择用户并设置角色（负责人/成员）
4. 设置是否为主团队
5. 点击「保存」

### 2.3 设置销售目标

#### 公司目标

1. 进入「目标管理」→「新建目标」
2. 选择目标类型：公司目标
3. 设置目标期间：年度/季度/月度
4. 填写各项目标指标：
   - 销售额目标
   - 回款目标
   - 新客户数目标
   - 线索数目标
   - 商机数目标
   - 签单数目标
5. 点击「保存」

#### 目标分解

**自动分解**
1. 选择公司或团队目标
2. 点击「自动分解」
3. 选择分解方法：
   - 平均分配：按团队/人员数量平均分配
   - 按比例：根据历史业绩按比例分配
4. 点击「确认分解」

**手动分解**
1. 选择公司或团队目标
2. 点击「手动分解」
3. 为每个团队/个人设置目标值
4. 点击「确认分解」

## 三、核心功能详解

### 3.1 组织架构管理

#### 团队层级结构

支持多级团队组织架构：

```
销售总部
├── 华东大区
│   ├── 江苏团队
│   └── 浙江团队
├── 华南大区
│   ├── 广东团队
│   └── 福建团队
└── 华北大区
    ├── 北京团队
    └── 河北团队
```

#### 团队类型

- **区域团队**：按地理区域划分（如华东团队）
- **产品团队**：按产品线划分（如智能硬件团队）
- **行业团队**：按行业划分（如制造业团队）

#### 一人多队

系统支持一个销售人员同时属于多个团队，例如：

- 张三属于"华东团队"（主团队）
- 张三属于"大客户团队"（兼职）

### 3.2 目标管理

#### 目标层级

1. **公司目标**：公司整体销售目标
2. **团队目标**：团队销售目标（由公司目标分解）
3. **个人目标**：个人销售目标（由团队目标分解）

#### 目标指标

- **销售额目标**：目标销售金额
- **回款目标**：目标回款金额
- **新客户数目标**：新增客户数量
- **线索数目标**：获取线索数量
- **商机数目标**：创建商机数量
- **签单数目标**：签约订单数量

#### 目标期间

- **年度目标**：整年目标
- **季度目标**：Q1、Q2、Q3、Q4
- **月度目标**：1-12月

### 3.3 目标分解流程

#### 自动分解示例

**场景**：公司年度目标 1000万，分解到3个区域团队

```
公司目标：1000万
↓ 自动平均分解
├── 华东团队：333.33万
├── 华南团队：333.33万
└── 华北团队：333.33万
```

#### 手动分解示例

**场景**：根据团队实力手动分解

```
公司目标：1000万
↓ 手动分解
├── 华东团队：400万（40%）
├── 华南团队：350万（35%）
└── 华北团队：250万（25%）
```

#### 二级分解

```
华东团队目标：400万
↓ 分解到个人
├── 张三：150万
├── 李四：120万
├── 王五：80万
└── 赵六：50万
```

### 3.4 目标跟踪

#### 实时更新

- 系统自动统计实际完成情况
- 自动计算完成率
- 实时更新排名

#### 完成率计算

```
完成率 = (实际销售额 / 目标销售额) × 100%
```

### 3.6 销售区域管理

#### 区域层级

```
中国区

│   │   └── 苏州市
```

3. 分配负责团队
4. 指定区域负责人

## 四、最佳实践

### 4.1 目标设定原则

1. **SMART原则**
   - Specific（具体的）
   - Measurable（可衡量的）
   - Achievable（可实现的）
   - Relevant（相关的）
   - Time-bound（有时限的）

2. **目标合理性**
   - 参考历史数据
   - 考虑市场环境
   - 结合团队能力

### 4.2 分解策略

1. **公司→团队分解**
   - 考虑团队规模
   - 参考历史业绩
   - 平衡区域难度

2. **团队→个人分解**
   - 考虑个人能力
   - 参考历史业绩
   - 保留部分弹性

### 4.3 跟踪频率

- **月度目标**：每周跟踪
- **季度目标**：每两周跟踪
- **年度目标**：每月跟踪

### 4.4 激励机制

1. **目标完成奖励**
   - 达成100%：基础奖金
   - 达成110%：超额奖金
   - 达成120%：卓越奖金

2. **排名奖励**
   - 季度冠军：额外奖励
   - 年度冠军：晋升机会

## 五、常见问题

### Q1: 如何调整已设定的目标？

A: 进入目标详情页，点击「编辑」，修改目标值后保存。如果已经分解，建议重新分解或手动调整子目标。

### Q2: 目标分解后发现不合理怎么办？

A: 可以删除分解的子目标，然后重新分解。或者直接修改子目标的值。

### Q3: 一个人可以同时属于多个团队吗？

A: 可以。系统支持一人多队，但需要指定一个主团队，用于业绩归属统计。

### Q4: 如何查看历史目标完成情况？

A: 进入「目标管理」，选择历史期间（如2025年），即可查看历史目标和完成情况。

### Q5: 完成率是如何计算的？

A: 完成率 = (实际完成值 / 目标值) × 100%。系统会根据实际销售额自动计算。

# 账户锁定用户手册

## 什么是账户锁定？

为了保护您的账户安全，系统实施了账户锁定机制。当检测到多次登录失败时，系统会自动锁定账户，防止恶意的暴力破解攻击。

## 锁定规则

- **失败次数阈值**: 连续5次登录失败
- **锁定时长**: 15分钟
- **验证码触发**: 3次失败后需要输入验证码（即将推出）

## 账户被锁定怎么办？

### 方法1：等待自动解锁（推荐）

账户锁定后，系统会在15分钟后自动解锁。您可以：

1. 等待15分钟
2. 重新尝试登录

系统会显示剩余锁定时间，例如：
```
账户已锁定，请在14:30后重试
```

### 方法2：联系管理员手动解锁

如果您急需访问系统，可以：

1. 联系系统管理员
2. 提供您的用户名和紧急原因
3. 管理员会在后台手动解锁您的账户

**联系方式：**
- 企业微信：@系统管理员
- 邮箱：admin@yourcompany.com
- 内线：8888

## 如何避免账户被锁定？

### ✅ 正确做法

1. **记住密码**: 使用密码管理器
2. **密码重置**: 忘记密码立即联系管理员重置，不要盲目尝试
3. **验证登录**: 登录前确认用户名和密码正确

### ❌ 错误做法

1. **反复尝试**: 不确定密码时反复尝试不同组合
2. **自动填充**: 使用过期的浏览器自动填充密码
3. **分享账号**: 将账号分享给他人可能导致误操作

## 常见问题

### Q1: 我确定密码正确，为什么还是失败？

**可能原因：**
- 大小写错误（密码区分大小写）
- 输入法状态（全角/半角、中英文）
- 前后空格
- 密码已被管理员重置

**解决方法：**
1. 在记事本中输入密码查看是否正确
2. 关闭输入法后重新输入
3. 联系管理员确认账户状态

### Q2: 我刚登录成功，为什么又被锁定？

账户锁定是基于**失败次数累计**的，如果：
- 您之前有过失败尝试
- 有人尝试登录您的账户
- 您在多个设备上同时登录失败

都可能触发锁定。

### Q3: 能否关闭我的账户锁定功能？

**不能**。账户锁定是系统安全策略的一部分，无法为单个用户关闭。这是为了保护所有用户的账户安全。

### Q4: 为什么错误消息是"用户名或密码错误"而不是"密码错误"？

这是**安全设计**。系统不会告诉攻击者用户名是否存在，防止账户枚举攻击。

## 安全提示

### 🔒 密码安全

- **长度**: 至少8位
- **复杂度**: 包含大小写字母、数字、特殊符号
- **唯一性**: 不要重用其他网站的密码
- **更新**: 每3个月更换一次

### 🚨 异常警报

如果您收到以下通知，请立即联系管理员：

- "您的账户已被锁定"（但您没有尝试登录）
- "检测到异常登录"
- "密码已重置"（但您没有申请）

这可能意味着有人正在尝试入侵您的账户。

## 系统日志

所有登录尝试（成功和失败）都会被记录，包括：
- 时间
- IP地址
- 设备信息

如需查看您的登录历史，可以：
1. 登录系统
2. 进入"个人中心" → "安全设置" → "登录历史"

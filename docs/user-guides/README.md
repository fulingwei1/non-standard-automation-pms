## 最佳实践

### ECN 管理最佳实践

1. **尽早分析影响** - ECN 批准后立即进行物料影响分析
2. **准确记录成本** - 每笔成本发生后及时录入
3. **保持信息同步** - 确保相关人员都订阅了通知
4. **定期跟踪进度** - 每周检查一次 ECN 执行进度
5. **善用成本预警** - 设置成本预警阈值

### 物料管理最佳实践

1. **每天查看进度** - 项目成员每天查看物料进度
2. **订阅关键通知** - 至少订阅齐套率预警/关键物料到货/缺料预警
3. **及时处理缺料** - 紧急缺料 24 小时内处理
4. **定期同步齐套率** - 每周一上午手动同步一次
5. **善用导出功能** - 重要节点导出报告作为会议材料

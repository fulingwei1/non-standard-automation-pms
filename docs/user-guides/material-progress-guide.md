# 项目物料进度跟踪使用指南

## 功能概述

项目成员可以随时查看物料进度，了解：
- 📊 项目整体物料齐套率
- 📦 每个 BOM 的齐套状态
- ⚠️ 缺料明细和影响
- 📅 预计到货日期
- 🔔 物料进度通知订阅

---

## 核心功能

### 1. 项目物料进度总览

**入口**: 项目管理 → 选择项目 → 物料进度

**查看内容**:
- 项目整体齐套率（0-100%）
- 物料状态（待采购/采购中/部分到货/齐套/缺料）
- 关键物料清单（影响交付的物料）
- 近 30 天齐套率趋势
- 预计齐套日期

**状态说明**:
| 状态 | 齐套率 | 说明 |
|------|--------|------|
| 待采购 | 0% | 还未开始采购 |
| 采购中 | 1-89% | 部分物料已下单 |
| 部分到货 | 90-94% | 大部分物料已到货 |
| 齐套 | 95-99% | 基本齐套 |
| 齐套 | 100% | 完全齐套 |
| 缺料 | - | 有关键物料缺失 |

**API**: `GET /api/v1/projects/{id}/material-progress`

---

### 2. BOM 物料明细进度

**入口**: 项目物料进度 → BOM 明细

**查看内容**:
- 每个 BOM 的齐套状态
- 每个物料的进度：
  - 需求数量
  - 已到货数量
  - 在途数量
  - 缺料数量
- 供应商交货进度
- 预计到货日期

**操作**:
- 点击物料可查看详细信息
- 点击供应商可查看供应商联系方式
- 点击采购订单可查看订单详情

**API**: `GET /api/v1/projects/{id}/bom-progress`

---

### 3. 缺料跟踪看板

**入口**: 项目物料进度 → 缺料跟踪

**查看内容**:
- 缺料清单（物料/缺料数量/影响）
- 缺料处理进度：
  - 已催货
  - 供应商回复
  - 预计到货
- 缺料对交付的影响天数
- 责任人跟踪

**缺料等级**:
| 等级 | 说明 | 处理要求 |
|------|------|----------|
| 紧急 | 影响关键路径 | 24 小时内解决 |
| 高 | 影响交付日期 | 3 天内解决 |
| 中 | 可能影响交付 | 1 周内解决 |
| 低 | 不影响交付 | 2 周内解决 |

**API**: `GET /api/v1/projects/{id}/shortage-tracker`

---

### 4. 物料进度通知订阅

**入口**: 项目物料进度 → 通知订阅

**可订阅的通知**:
- 齐套率变化通知（如从 80%→90%）
- 关键物料到货通知
- 缺料预警通知
- 供应商发货通知

**通知渠道**:
- 站内通知（必选）
- 邮件通知（可选）
- 短信通知（可选）
- 企业微信（可选）

**操作步骤**:
1. 进入项目物料进度页面
2. 点击右上角"通知订阅"
3. 选择要接收的通知类型
4. 选择通知渠道
5. 保存订阅设置

**API**: `POST /api/v1/projects/{id}/material-progress/subscribe`

## 使用场景

### 场景 1: 项目经理查看项目物料进度

**背景**: 项目经理需要了解项目物料齐套情况

**步骤**:
1. 打开项目管理页面
2. 选择负责的项目
3. 点击"物料进度"标签
4. 查看齐套率和缺料情况
5. 如有缺料，查看缺料跟踪看板
6. 必要时催货或调整计划

**输出**:
- 项目齐套率报告
- 缺料清单
- 预计齐套日期

---

### 场景 2: 采购员跟踪物料到货

**背景**: 采购员需要跟踪负责物料的到货情况

**步骤**:
1. 打开物料跟踪页面
2. 筛选"我负责的物料"
3. 查看在途物料状态
4. 关注预计到货日期
5. 如有延期，联系供应商

**输出**:
- 在途物料清单
- 供应商交货进度
- 延期预警

---

### 场景 3: 项目成员订阅物料通知

**背景**: 项目成员希望及时收到物料进度通知

**步骤**:
1. 进入项目物料进度页面
2. 点击"通知订阅"
3. 勾选"齐套率变化"
4. 勾选"关键物料到货"
5. 勾选"缺料预警"
6. 选择通知渠道（站内 + 企业微信）
7. 保存

**效果**:
- 齐套率每变化 10% 收到通知
- 关键物料到货立即收到通知
- 发现缺料立即收到预警

---

### 场景 4: 缺料处理

**背景**: 发现关键物料缺料，需要紧急处理

**步骤**:
1. 打开缺料跟踪看板
2. 找到缺料物料
3. 查看缺料数量和影响
4. 点击"催货"按钮
5. 选择催货方式（电话/邮件/现场）
6. 记录催货结果
7. 更新预计到货日期

**输出**:
- 催货记录
- 供应商回复
- 更新后的预计到货日期

## 齐套率计算逻辑

### 计算公式

```
齐套率 = (已到齐的 BOM 行数 / 总需采购行数) × 100%
```

### 到齐判定

| 物料状态 | 是否计入已到齐 |
|----------|----------------|
| 已入库 | ✅ 是 |
| 在途（预计到货≤计划需求日期） | ✅ 是 |
| 在途（预计到货>计划需求日期） | ❌ 否 |
| 未下单 | ❌ 否 |
| 缺料 | ❌ 否 |

### 权重调整

关键物料（影响关键路径）权重 × 1.5

## 定时同步

### 自动同步

系统**每小时整点**自动同步所有项目的齐套率：
- 计算最新齐套率
- 更新项目健康度
- 识别齐套率显著变化的项目
- 记录同步日志

### 手动同步

如需立即同步：
1. 进入项目物料进度页面
2. 点击右上角"同步"按钮
3. 等待同步完成

**API**: `POST /api/v1/material/sync-kitting-rate`

## 常见问题

### Q1: 齐套率为什么没有实时更新？

**A**: 齐套率每小时同步一次，如需立即更新：
- 点击"同步"按钮手动触发
- 等待下一次定时同步（整点）

---

### Q2: 如何识别关键物料？

**A**: 关键物料会自动标记：
- 影响关键路径的物料
- 长周期物料（采购周期>30 天）
- 单一来源物料

在 BOM 明细中会显示"关键"标签。

---

### Q3: 缺料影响天数如何计算？

**A**: 系统自动计算：
```
影响天数 = 预计到货日期 - 计划需求日期
```

如预计到货日期晚于计划需求日期，则为正数（延期）。

---

### Q4: 如何批量查看多个项目的物料进度？

**A**: 使用物料分析页面：
1. 进入物料分析页面
2. 选择多个项目
3. 查看汇总齐套率
4. 导出 Excel 报告

---

### Q5: 通知太多怎么办？

**A**: 可以调整订阅设置：
1. 进入通知订阅页面
2. 取消不重要的通知类型
3. 调整齐套率变化阈值（如从 5% 改为 10%）
4. 只保留关键通知

## 最佳实践

### 1. 每天查看物料进度

建议项目成员**每天上班后**查看一次物料进度，及时发现问题。

### 2. 订阅关键通知

至少订阅：
- 齐套率<90% 预警
- 关键物料到货
- 缺料预警

### 3. 及时处理缺料

发现缺料后：
- 紧急缺料：24 小时内处理
- 高优先级：3 天内处理
- 中优先级：1 周内处理

### 4. 定期同步齐套率

建议**每周一上午**手动同步一次齐套率，确保数据准确。

### 5. 善用导出功能

重要节点（如周会/月会）前，导出物料进度报告作为会议材料。

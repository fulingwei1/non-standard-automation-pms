> **版本**: v1.0  
> **最后更新**: 2025-01-XX  
> **适用系统**: 非标自动化项目管理系统

---

## 一、功能概述

项目管理模块是系统的核心模块，用于管理项目从需求进入（S1）到质保结项（S9）的完整生命周期。

### 主要功能

1. **项目创建与编辑** - 创建新项目，编辑项目信息
2. **阶段推进管理** - 通过阶段门校验推进项目阶段
3. **健康度监控** - 实时监控项目健康状态
4. **机台管理** - 管理项目下的设备/机台
5. **里程碑管理** - 设置和跟踪项目关键节点
6. **项目成员管理** - 分配和管理项目团队成员
7. **项目模板** - 使用模板快速创建项目

### 项目生命周期

项目遵循 9 个阶段的完整生命周期：

| 阶段 | 名称 | 说明 |
|------|------|------|
| S1 | 需求进入 | 项目需求初步确认 |
| S2 | 需求澄清 | 需求详细分析和确认 |
| S3 | 立项评审 | 项目立项和评审 |
| S4 | 方案设计 | 技术方案设计 |
| S5 | 采购制造 | 物料采购和加工制造 |
| S6 | 装配联调 | 设备装配和联调测试 |
| S7 | 出厂验收 | 工厂验收（FAT） |
| S8 | 现场交付 | 现场安装和验收（SAT） |
| S9 | 质保结项 | 质保期结束，项目结项 |

### 2.1 登录系统

## 三、项目创建

### 3.1 创建新项目

1. 在项目列表页面，点击右上角"新建项目"按钮
2. 系统会打开项目创建表单（分步骤表单）

#### 步骤 1: 基本信息

填写以下必填信息：
- **项目编码**: 系统自动生成，格式为 `PJ + 日期 + 序号`，也可手动修改
- **项目名称**: 输入项目完整名称
- **项目简称**: 输入项目简称（可选）
- **项目类型**: 选择"固定价格"或"工时材料"
- **设备类型**: 选择设备类型（如：ICT测试设备、FCT测试设备等）

**提示**: 
- 项目编码需要唯一，系统会自动检查
- 如果编码重复，系统会提示错误

#### 步骤 2: 客户信息

- **客户**: 从下拉列表选择客户，或输入关键词搜索
- 选择客户后，系统会自动填充：
  - 客户联系人
  - 联系电话
  - 客户地址

**提示**: 
- 如果客户不存在，需要先在"客户管理"模块创建客户

#### 步骤 3: 财务信息

- **合同金额**: 输入项目合同总金额
- **预算金额**: 输入项目预算金额
- **项目经理**: 从下拉列表选择项目经理
  - 系统会显示项目经理当前负责的项目数量
  - 建议选择工作负载较低的项目经理

#### 步骤 4: 时间节点

- **计划开始日期**: 选择项目计划开始日期
- **计划结束日期**: 选择项目计划结束日期
- **实际开始日期**: 项目实际开始日期（可选）

**提示**: 
- 结束日期必须晚于开始日期
- 系统会检查交期冲突（如果启用）

### 3.2 使用模板创建项目

1. 在项目创建表单中，系统会自动推荐合适的项目模板
2. 点击模板卡片，查看模板详情
3. 点击"应用模板"按钮，系统会自动填充模板中的配置
4. 根据需要修改字段值
5. 点击"提交"完成创建

**提示**: 
- 模板推荐基于项目类型、设备类型、客户历史项目等因素
- 应用模板后仍可手动修改所有字段

### 3.3 保存草稿

在填写表单过程中，可以随时点击"保存草稿"按钮，系统会将当前填写的内容保存到本地。

下次打开创建表单时，系统会自动加载草稿内容。

## 四、项目编辑

### 4.1 编辑项目信息

1. 在项目列表页面，找到要编辑的项目
2. 点击项目名称或"编辑"按钮，进入项目详情页
3. 点击"编辑"按钮，打开编辑表单
4. 修改需要更新的字段
5. 点击"保存"完成更新

**注意事项**:
- 项目编码创建后不可修改
- 修改项目状态会自动触发健康度重新计算
- 修改项目阶段需要满足阶段门校验条件

## 五、项目查看

### 5.1 项目列表

项目列表页面显示所有项目，支持：

- **筛选**: 按阶段、状态、健康度、客户、项目经理筛选
- **搜索**: 输入关键词搜索项目名称或编码
- **排序**: 点击表头排序（按创建时间、交期等）
- **分页**: 使用底部分页控件翻页

### 5.2 项目详情

点击项目名称进入项目详情页，包含以下标签页：

#### 概览（Overview）

显示项目基本信息：
- 项目基本信息卡片
- 财务信息卡片
- 时间节点卡片
- 阶段门校验面板（如果当前阶段有前置门）
- 快速操作面板

#### 时间线（Timeline）

显示项目关键事件时间线：
- 状态变更记录
- 阶段推进记录
- 里程碑完成记录
- 文档上传记录

支持：
- 按事件类型筛选
- 关键词搜索
- 时间范围筛选

#### 机台（Machines）

显示项目下的所有设备/机台：
- 机台列表
- 机台状态
- 机台详情

#### 里程碑（Milestones）

显示项目里程碑：
- 里程碑列表
- 计划日期 vs 实际日期
- 里程碑状态（未开始/进行中/已完成/已逾期）

#### 团队（Team）

显示项目团队成员：
- 成员列表
- 成员角色
- 投入比例
- 工作时间

## 六、阶段推进

### 6.1 查看阶段门校验结果

1. 在项目详情页的"概览"标签页
2. 找到"阶段门校验"面板
3. 查看当前阶段到下一阶段的校验结果

阶段门校验面板显示：
- 阶段门名称（如：G1: 需求进入→需求澄清）
- 校验进度（百分比）
- 校验条件列表（每个条件的通过/失败状态）
- 缺失项列表
- 改进建议

### 6.2 推进项目阶段

当阶段门校验全部通过后：

1. 在"阶段门校验"面板底部，点击"推进至 [目标阶段] 阶段"按钮
2. 系统会弹出确认对话框
3. 输入推进原因（可选）
4. 点击"确认"完成推进

**注意事项**:
- 只有满足所有阶段门条件才能推进
- 管理员可以跳过阶段门校验（不推荐）
- 阶段推进会记录到项目时间线

### 6.3 阶段门说明

| 阶段门 | 从阶段 | 到阶段 | 主要校验条件 |
|--------|--------|--------|--------------|
| G1 | S1 | S2 | 客户信息齐全、需求采集表完整 |
| G2 | S2 | S3 | 需求规格书已确认、验收标准明确 |
| G3 | S3 | S4 | 立项评审通过、预算已批准 |
| G4 | S4 | S5 | 方案设计完成、BOM已确认 |
| G5 | S5 | S6 | 物料齐套、加工完成 |
| G6 | S6 | S7 | 装配完成、联调通过 |
| G7 | S7 | S8 | FAT验收通过、包装完成 |
| G8 | S8 | S9 | SAT验收通过、质保期结束 |

## 七、健康度监控

### 7.1 健康度等级

项目健康度分为 4 个等级：

- **H1 - 正常（绿色）**: 项目按计划进行，无风险
- **H2 - 有风险（黄色）**: 存在风险因素，需要关注
- **H3 - 阻塞（红色）**: 项目受阻，需要立即处理
- **H4 - 已完结（灰色）**: 项目已结项或取消

### 7.2 查看健康度详情

1. 在项目详情页，点击健康度标签（如：H2）
2. 系统会显示健康度详情对话框，包含：
   - 健康度诊断说明
   - 影响因素列表
   - 改进建议

### 7.3 健康度影响因素

健康度由以下因素决定：

- **项目状态**: 阻塞状态（ST14/ST19）→ H3，整改状态（ST22/ST26）→ H2
- **交期临近**: 交期在 7 天内 → H2
- **逾期里程碑**: 有关键里程碑逾期 → H2
- **缺料预警**: 有严重缺料预警 → H3，有一般缺料预警 → H2
- **阻塞问题**: 有严重阻塞问题未解决 → H3
- **进度偏差**: 进度偏差超过 10% → H2

4. 点击"保存"完成更新

## 九、里程碑管理

### 9.1 创建里程碑

1. 在项目详情页的"里程碑"标签页
2. 点击"新建里程碑"按钮
3. 填写里程碑信息：
   - 里程碑名称
   - 计划日期
   - 是否关键里程碑
   - 里程碑描述
4. 点击"保存"完成创建

### 9.2 更新里程碑状态

1. 在里程碑列表中，找到要更新的里程碑
2. 点击"完成"按钮，或点击里程碑进入详情页编辑
3. 更新实际完成日期和状态
4. 点击"保存"完成更新

**提示**: 
- 关键里程碑逾期会影响项目健康度
- 里程碑完成会记录到项目时间线

## 十二、常见问题

### Q1: 如何修改项目编码？

A: 项目编码创建后不可修改。如果需要修改，请联系系统管理员。

### Q2: 为什么无法推进项目阶段？

A: 可能原因：
1. 阶段门校验未通过，请查看"阶段门校验"面板中的缺失项
2. 没有推进权限，请联系项目经理或系统管理员

### Q3: 项目健康度如何自动更新？

A: 健康度在以下情况自动更新：
- 项目状态更新时
- 项目阶段推进时
- 定时任务（每小时自动计算）

也可以手动触发计算：在项目详情页点击健康度标签，然后点击"重新计算"。

### Q4: 如何使用项目模板？

A: 
1. 创建项目时，系统会自动推荐模板
2. 点击模板卡片查看详情
3. 点击"应用模板"自动填充配置
4. 根据需要修改字段值

### Q5: 如何查看项目历史记录？

A: 在项目详情页的"时间线"标签页，可以查看：
- 状态变更历史
- 阶段推进历史
- 里程碑完成记录
- 文档上传记录

支持按事件类型和时间范围筛选。

### Q6: 项目删除后可以恢复吗？

A: 项目采用软删除，删除后不会真正删除数据。请联系系统管理员恢复。

## 十三、操作技巧

### 技巧 1: 使用筛选快速查找项目

在项目列表页面，使用多个筛选条件组合，可以快速定位到目标项目。

例如：筛选"阶段=S5" + "健康度=H3"，可以快速找到装配阶段中阻塞的项目。

### 技巧 2: 使用模板提高创建效率

对于相似类型的项目，建议创建项目模板。创建新项目时直接应用模板，可以大幅提高创建效率。

### 技巧 3: 关注健康度预警

定期查看项目健康度，特别是 H2 和 H3 状态的项目，及时处理风险因素，避免项目延期。

### 技巧 4: 使用时间线追踪项目进展

在项目详情页的"时间线"标签页，可以清晰地看到项目的关键事件历史，帮助了解项目进展。

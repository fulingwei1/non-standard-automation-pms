## 🚀 核心功能展示

### 1. 客户画像分析

```bash
curl -X POST "http://localhost:8000/api/v1/presale/ai/analyze-customer-profile" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": 12345,
    "communication_notes": "客户是技术总监，关注系统架构和稳定性..."
  }'
```

**输出**:
```json
{
  "customer_type": "technical",
  "focus_points": ["quality", "delivery"],
  "decision_style": "rational",
  "ai_analysis": "典型的技术型客户，建议从技术架构切入..."
}
```

---

### 2. 智能话术推荐

支持 **6大场景**:
- 首次接触 → 需求挖掘 → 方案讲解 → 价格谈判 → 异议处理 → 成交

**示例话术**（首次接触-技术型）:
> "您好，我是XX公司的技术顾问。了解到贵司在寻找XXX解决方案，我们在该领域有成熟的技术架构和实施经验，方便和您深入探讨技术细节吗？"

---

### 3. 异议处理引擎

**异议**: "价格太高"

**AI推荐策略**:
1. 强调价值和ROI
2. 提供TCO对比
3. 分期付款方案

**推荐话术**:
- "价格确实是考量因素，但更重要的是投入产出比..."
- "算上实施成本和维护成本，我们的总拥有成本最低..."
- "我们可以提供分期付款，首付只需XX%..."

---

### 4. 销售进程指导

**输入**: 当前销售情况描述

**输出**:
- 当前阶段识别
- 3-5个下一步行动
- 关键里程碑
- 潜在风险预警
- 预计时间线

## 🎓 使用指南

### 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境
cp .env.example .env
# 编辑.env：配置数据库和API密钥

# 3. 初始化数据库
mysql -u root -p presale_db < migrations/001_create_tables.sql

# 4. 导入话术库
cd data && python3 import_seeds.py && cd ..

# 5. 启动服务
python3 -m app.main

# 6. 访问API文档
# http://localhost:8000/docs
```

### 运行测试

```bash
# 运行所有测试
pytest -v

# 生成覆盖率报告
pytest --cov=app --cov-report=html
```

```bash
python3 verify_deliverables.py
```

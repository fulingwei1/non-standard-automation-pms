## 产品概述

### 什么是AI智能话术推荐引擎？

AI智能话术推荐引擎是一款基于人工智能的销售辅助工具，帮助销售人员：
- **精准识别客户画像**：自动分析客户类型、关注点、决策风格
- **智能推荐话术**：根据场景和客户特征推荐最合适的销售话术
- **处理客户异议**：针对常见异议提供应对策略和话术
- **指导销售进程**：分析当前进度，提供下一步行动建议

### 核心价值

✅ **提升成单率**：精准话术提升客户沟通效率，成单率提升20-30%  
✅ **缩短销售周期**：AI指导快速推进销售进程，周期缩短15-25%  
✅ **新人快速上手**：话术库和AI建议降低新人培训成本50%  
✅ **标准化流程**：统一话术标准，提升团队整体水平

### 步骤2：第一次使用

**场景**：销售小王刚接触一位新客户，通过电话沟通了30分钟。

```python
import requests

# 1. 分析客户画像
response = requests.post(
    "http://localhost:8000/api/v1/presale/ai/analyze-customer-profile",
    json={
        "customer_id": 12345,
        "presale_ticket_id": 101,
        "communication_notes": """
        客户是一家制造企业的技术总监，对我们的产品架构很感兴趣。
        沟通中多次提到系统稳定性和扩展能力，询问了API对接方式。
        对价格敏感度不高，但强调项目交付周期要短。
        """
    }
)
profile = response.json()
print(f"客户类型：{profile['customer_type']}")  # technical
print(f"关注点：{profile['focus_points']}")  # ['quality', 'delivery']

# 2. 获取推荐话术
response = requests.post(
    "http://localhost:8000/api/v1/presale/ai/recommend-sales-script",
    json={
        "presale_ticket_id": 101,
        "scenario": "needs_discovery",
        "customer_profile_id": profile['id']
    }
)
scripts = response.json()
print("推荐话术：")
for script in scripts['recommended_scripts']:
    print(f"- {script}")
```

## 核心功能详解

### 1. 客户画像分析

#### 功能说明
系统基于沟通记录，自动识别：
- **客户类型**：technical（技术型）、commercial（商务型）、decision_maker（决策型）、mixed（混合型）
- **关注点**：price（价格）、quality（质量）、delivery（交期）、service（服务）
- **决策风格**：rational（理性）、emotional（感性）、authoritative（权威）

#### 使用场景
- 首次接触客户后
- 客户需求发生变化时
- 准备重要会议前
- 定期更新客户档案

#### 操作示例

```bash
curl -X POST "http://localhost:8000/api/v1/presale/ai/analyze-customer-profile" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": 12345,
    "communication_notes": "客户是XX公司CTO，关注技术架构和系统稳定性..."
  }'
```

#### 输出解读

```json
{
  "customer_type": "technical",  // 技术型客户
  "focus_points": ["quality", "delivery"],  // 关注质量和交期
  "decision_style": "rational",  // 理性决策
  "ai_analysis": "典型的技术型客户，决策理性，建议从技术架构、案例经验切入..."
}
```

**使用建议**：
- ✅ 提供完整的沟通记录（至少200字）
- ✅ 包含客户的具体问题和关注点
- ✅ 定期更新，保持画像准确性
- ❌ 避免只有简单描述

### 2. 场景化话术推荐

#### 6大销售场景

| 场景 | 说明 | 典型问题 |
|------|------|----------|
| **首次接触** (first_contact) | 建立初步联系，引起兴趣 | 如何开场？如何介绍产品？ |
| **需求挖掘** (needs_discovery) | 深入了解客户需求 | 如何提问？如何挖掘痛点？ |
| **方案讲解** (solution_presentation) | 展示解决方案价值 | 如何展示？如何突出优势？ |
| **价格谈判** (price_negotiation) | 价格协商，促成交易 | 如何报价？如何应对议价？ |
| **异议处理** (objection_handling) | 处理客户疑虑和异议 | 如何应对？如何说服？ |
| **成交** (closing) | 推动签约，锁定订单 | 如何临门一脚？如何催单？ |

#### 操作示例

```python
# 获取"首次接触"场景话术
response = requests.post(
    "http://localhost:8000/api/v1/presale/ai/recommend-sales-script",
    json={
        "presale_ticket_id": 101,
        "scenario": "first_contact",
        "customer_profile_id": 1,
        "context": "通过行业展会认识，客户主动要了名片"
    }
)
```

#### 输出解读

系统会返回：
1. **推荐话术**（3-5条）：可直接使用或改编
2. **应对策略**：整体思路和注意事项
3. **成功案例**：类似场景的成功经验

**使用技巧**：
- 💡 不要照本宣科，根据实际情况调整
- 💡 结合客户画像，选择最匹配的话术
- 💡 记录使用效果，反馈系统优化

### 3. 异议处理建议

#### 常见异议类型

| 异议类型 | 应对策略要点 |
|---------|-------------|
| 价格太高 | 强调价值和ROI，提供TCO对比 |
| 技术不成熟 | 展示成功案例，提供试用机会 |
| 竞品更好 | 突出差异化优势，客观对比 |
| 暂时不需要 | 创造紧迫感，提供低门槛方案 |
| 预算不足 | 分期付款，调整方案范围 |
| 决策周期长 | 协助准备材料，加速流程 |
| 兼容性问题 | 提供技术方案，承诺支持 |
| 数据安全担忧 | 强调安全措施，支持私有化 |

#### 操作示例

```bash
curl -X POST "http://localhost:8000/api/v1/presale/ai/handle-objection" \
  -H "Content-Type: application/json" \
  -d '{
    "presale_ticket_id": 102,
    "objection_type": "价格太高",
    "context": "客户说我们的报价比竞品A高20%"
  }'
```

#### 处理技巧

**LAER异议处理法**：
1. **Listen（倾听）**：认真听客户说完，不要打断
2. **Acknowledge（认同）**："我理解您的顾虑..."
3. **Explore（探索）**：深入了解异议背后的真实原因
4. **Respond（回应）**：使用推荐话术应对

**示例对话**：
```
客户："你们的价格比XX公司高很多。"

❌ 错误应对：
"我们不贵啊，XX公司质量不行..."

✅ 正确应对（使用AI推荐话术）：
"我完全理解您对价格的关注（认同）。能具体说说您对比的是哪个方案吗（探索）？
价格确实是重要考量因素，但更重要的是投入产出比。我们的方案能带来XX%的效率提升，
算上实施成本和长期维护，总拥有成本其实更低。我给您做个详细对比？（回应）"
```

## 常见问题

### Q1: AI推荐的话术能直接用吗？

**A**: 不建议完全照搬，原因：
1. 每个客户情况独特，需要个性化调整
2. 话术需要符合你的表达习惯
3. 实时对话需要灵活应变

**建议做法**：
- ✅ 理解话术背后的策略和逻辑
- ✅ 用自己的语言重新表达
- ✅ 根据对话情境灵活调整
- ✅ 融入公司和产品的具体信息

---

### Q2: 客户画像不准怎么办？

**A**: 画像准确性取决于输入质量：

**提升准确性的方法**：
1. **提供充分信息**：至少200字以上的沟通记录
2. **包含关键细节**：客户的问题、关注点、态度
3. **多次互动后更新**：随着了解深入，定期更新画像
4. **人工校准**：AI分析后人工review，必要时调整

**画像分析checklist**：
- [ ] 客户的角色/职位
- [ ] 主要关注的问题/需求
- [ ] 沟通中的态度（积极/谨慎/抗拒）
- [ ] 决策风格表现
- [ ] 预算和时间敏感度

---

### Q3: 如何评估话术效果？

**A**: 使用反馈机制持续优化：

```python
# 每次使用话术后记录效果
requests.post(
    "/api/v1/presale/ai/script-feedback",
    json={
        "script_id": 101,
        "is_effective": True,  # 或False
        "feedback_notes": "客户对这个话术反应很好，成功推进到下一阶段"

```

**评估指标**：
- 客户响应积极度（1-5分）
- 是否推进到下一阶段
- 客户提出的新问题/异议
- 最终成交与否

## 话术使用技巧

### 技巧1：3秒法则

**原理**：话术开头3秒决定客户是否愿意继续听

**优化方法**：
```
❌ 差的开场："您好，我是XX公司的销售..."
✅ 好的开场："我看到贵司最近在XXX项目上有进展，恭喜！顺便想和您交流..."

分析：
- 差的开场：典型销售套路，客户防备心强
- 好的开场：展示了解、给予肯定、自然切入
```

**AI话术优化**：
- 找到AI推荐话术中的"价值点"
- 把价值点前置到开头3秒
- 用问句或利益点吸引注意

---

### 技巧2：FABE销售法

将AI推荐的话术改造成FABE结构：

- **F (Feature)**: 特征 - 产品是什么
- **A (Advantage)**: 优势 - 比竞品好在哪
- **B (Benefit)**: 利益 - 客户得到什么
- **E (Evidence)**: 证据 - 案例/数据支撑

**示例改造**：

AI原话术：
> "我们的系统支持私有化部署，数据完全在您的内网。"

改造后（FABE）：
> "我们的系统支持私有化部署（F-特征），这意味着所有数据完全在您的内网，不经过我们的服务器（A-优势）。对于贵司这样对数据安全要求高的金融企业，可以完全消除数据泄露的风险（B-利益）。XX银行就是采用这个方案，已安全运行2年，0事故（E-证据）。"

---

### 技巧3：故事化表达

**原理**：人脑对故事的记忆是数据的22倍

**改造方法**：
```
AI话术（数据型）：
"我们的方案能帮您节省30%的成本，提升20%的效率。"

改造为故事：
"去年XX公司也遇到和您类似的问题，运营成本居高不下。他们用了我们的方案后，
第一个月就发现流程环节从15个减少到10个，3个月后算总账，成本降了30%，
团队效率提升了20%。他们的运营总监和我说，早知道效果这么好，应该更早启动。"
```

**故事三要素**：
1. **场景相似**："XX公司也遇到和您类似的问题..."
2. **具体过程**："他们用了我们的方案后..."
3. **可感结果**："成本降了30%...运营总监说..."

---

### 技巧4：异议转化

**不要"解决"异议，而是"转化"异议**

**转化公式**：
```
客户异议 → 认同 → 重新定义 → 展示价值

示例：
客户："你们的产品太贵了。"

第1步-认同：
"我完全理解您对价格的关注，这确实是重要的考量因素。"

第2步-重新定义：
"不过我想换个角度，如果我们讨论的不是价格，而是投资回报，您觉得怎样？"

第3步-展示价值：
"我们的方案虽然初期投入XX万，但能帮您每年节省YY万，6个月回本。
更重要的是，这个方案能支撑您未来3年的业务增长，不用重复投资。
这样算下来，每年的实际成本只有ZZ万，比竞品方案还要低。"
```

### 技巧5：节奏控制

**销售对话的"快慢"节奏**：

| 阶段 | 节奏 | 话术特点 | AI辅助 |
|------|------|---------|--------|
| 开场 | 快 | 简短有力，3秒吸引 | 使用"首次接触"高成功率话术 |
| 需求挖掘 | 慢 | 多提问，多倾听 | 使用"需求挖掘"提问话术 |
| 方案讲解 | 中 | 讲+互动，确认理解 | 使用"方案讲解"结构化话术 |

**实战技巧**：
- 用停顿控制节奏："...（停3秒）您觉得呢？"
- 用问句放慢节奏："在这方面，您最关注什么？"
- 用总结加快节奏："那我们就这样定了，我安排商务准备合同？"

### 技巧6：个性化改编

**AI话术 + 你的风格 = 最佳效果**

**改编检查清单**：
- [ ] 符合我的表达习惯吗？
- [ ] 包含客户的具体信息吗？（公司名、行业、痛点）
- [ ] 体现了我们产品的独特价值吗？
- [ ] 有具体数据/案例支撑吗？
- [ ] 听起来真诚自然吗？

**示例改编**：

AI通用话术：
> "我们的产品在该架构下有深度优化，API对接简单。"

个性化改编：
> "王总，我注意到贵司用的是Java技术栈（客户具体信息），我们的产品专门针对Java做了深度优化，提供了完整的Spring Boot集成包（具体价值）。像XX公司，他们的工程师只用了3天就完成了全部对接（具体案例），这是他们的代码示例，可以发给您的技术团队参考。"

### 使用AI智能话术推荐引擎的5个关键原则

1. **理解 > 照搬**：理解话术背后的逻辑，而非机械使用
2. **个性化 > 标准化**：根据客户特点调整，避免千篇一律
3. **倾听 > 话术**：再好的话术也要基于充分倾听
4. **实践 > 理论**：多用多练，形成肌肉记忆
5. **反馈 > 完美**：记录效果，持续优化

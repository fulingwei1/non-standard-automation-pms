# 管理节律模板说明

本目录包含管理节律系统使用的Excel和Word模板文件。

## Excel模板

### 1. 战略会议模板.xlsx

用于战略层会议（季度战略研讨会、战略解码会等）。

**Sheet结构：**
- **Sheet1: 会议基本信息**
  - 会议名称、日期、时间、地点
  - 组织者、参会人员
  - 会议议程

- **Sheet2: 战略分析（五看三定）**
  - 看行业：行业趋势分析
  - 看市场：市场机会分析
  - 看客户：客户需求分析
  - 看竞争：竞争对手分析
  - 看自己：自身能力分析
  - 三定：定方向、定目标、定路径

- **Sheet3: 关键决策记录**
  - 决策事项
  - 决策人
  - 决策时间
  - 决策依据

- **Sheet4: 资源分配计划**
  - 预算调整
  - 资源重新分配
  - 人员调配

- **Sheet5: 行动项跟踪**
  - 行动项描述
  - 责任人
  - 截止日期
  - 状态跟踪

### 2. 经营分析会模板.xlsx

用于经营层会议（月度经营分析会、年度经营目标会等）。

**Sheet结构：**
- **Sheet1: 财务指标对比**
  - 收入：实际 vs 目标
  - 成本：实际 vs 预算
  - 利润：实际 vs 目标
  - 现金流：实际 vs 预测
  - 毛利率、净利润率

- **Sheet2: 差距分析**
  - 收入差距及原因
  - 成本超支分析
  - 利润差距分析
  - 根因追溯

- **Sheet3: 改善措施**
  - 改善措施清单
  - 责任人
  - 完成时间
  - 预期效果

- **Sheet4: 预算调整**
  - 预算调整项目
  - 调整金额
  - 调整原因
  - 审批状态

### 3. 运营例会模板.xlsx

用于运营层会议（周运营例会、项目复盘会等）。

**Sheet结构：**
- **Sheet1: 流程指标监控**
  - 订单交付周期
  - 生产良率
  - 招聘达成率
  - 其他关键流程指标

- **Sheet2: 项目进展**
  - 项目列表
  - 当前阶段
  - 进度百分比
  - 健康状态
  - 风险项

- **Sheet3: 异常处理**
  - 异常事件列表
  - 异常类型
  - 严重程度
  - 处理状态
  - 责任人

- **Sheet4: 改进计划**
  - 改进项
  - 改进措施
  - 预期效果
  - 完成时间

### 4. 日清会模板.xlsx

用于任务层会议（日清会、晨会等）。

**Sheet结构：**
- **Sheet1: 当日任务清单**
  - 任务描述
  - 责任人
  - 计划完成时间
  - 实际完成时间
  - 状态

- **Sheet2: 障碍记录**
  - 障碍描述
  - 影响任务
  - 紧急程度
  - 处理方案

- **Sheet3: 根因分析**
  - 问题描述
  - 5Why分析
  - 根本原因
  - 纠正措施

- **Sheet4: 明日计划**
  - 计划任务
  - 优先级
  - 预计工时
  - 资源需求

## 使用说明

1. **下载模板**：从系统下载对应层级的模板文件
2. **填写信息**：按照模板结构填写会议相关信息
3. **上传附件**：在创建或编辑会议时，上传填写好的模板文件
4. **系统同步**：系统会自动解析模板中的关键信息（如行动项）

## 模板更新

模板会根据管理节律配置自动更新。管理员可以在"节律配置"页面修改会议模板结构。

## 注意事项

- 请保持模板结构不变，以确保系统能正确解析
- 日期格式统一使用 YYYY-MM-DD
- 金额统一使用数字格式，保留两位小数
- 行动项必须包含：描述、责任人、截止日期

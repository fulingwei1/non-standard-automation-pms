# 工时报表模板说明

本目录包含工时相关报表的模板文件说明。实际报表由系统动态生成，这些模板仅作为格式参考。

## 报表类型

### 1. HR加班工资表 (hr_overtime_report)
**用途**: 人力资源部门计算加班工资

**包含字段**:
- 员工姓名
- 部门
- 正常工时
- 工作日加班工时
- 周末加班工时
- 节假日加班工时
- 工作日加班工资
- 周末加班工资
- 节假日加班工资
- 合计加班工资

**生成时机**: 每月1号自动生成上月报表

### 2. 财务成本核算表 (finance_cost_report)
**用途**: 财务部门核算项目成本

**包含字段**:
- 项目编号
- 项目名称
- 人员姓名
- 工作日期
- 工时
- 时薪
- 成本金额
- 工作内容

**生成时机**: 工时审批通过后立即生成，或每月1号批量生成

### 3. 研发费用核算表 (rd_cost_report)
**用途**: 研发部门核算研发费用

**包含字段**:
- 研发项目编号
- 研发项目名称
- 人员姓名
- 工作日期
- 工时
- 时薪
- 费用金额
- 工作内容

**生成时机**: 工时审批通过后立即生成，或每月1号批量生成

### 4. 项目工时报表 (project_timesheet_report)
**用途**: 项目进度查看和成本分析

**包含Sheet**:
- Sheet1: 人员贡献度统计
- Sheet2: 每日工时分布

**生成时机**: 实时生成，可按需导出

## 使用说明

所有报表通过API端点动态生成，无需手动填写模板。模板文件仅作为格式参考。

如需导出报表，请使用以下API端点：
- `GET /api/v1/timesheet/reports/hr` - 获取HR报表
- `GET /api/v1/timesheet/reports/finance` - 获取财务报表
- `GET /api/v1/timesheet/reports/rd` - 获取研发报表
- `GET /api/v1/timesheet/reports/project` - 获取项目报表

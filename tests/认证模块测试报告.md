## 关键问题汇总

### 🔴 严重问题（需立即修复）

1. **Token刷新功能失效**
   - **文件**: `tests/api/test_auth.py::test_refresh_token`
   - **问题**: Refresh token端点返回422而非200
   - **影响**: 用户无法刷新token，长时间使用需要重新登录
   - **修复建议**:
     ```python
     # 当前测试代码问题
     response = client.post(f"{settings.API_V1_PREFIX}/auth/refresh", headers=headers)

     # 应该传入refresh_token
     response = client.post(
         f"{settings.API_V1_PREFIX}/auth/refresh",
         json={"refresh_token": refresh_token}
     )
     ```

2. **账户锁定防护失效**
   - **文件**: `tests/integration/test_auth_lockout_integration.py::test_login_lockout_flow`
   - **问题**: 连续5次失败登录后返回401而非423
   - **影响**: 暴力破解防护可能失效
   - **修复建议**: 检查`AccountLockoutService.record_failed_login`是否正确递增失败次数

3. **登出功能异常**
   - **文件**: `tests/api/test_auth_security.py::TestLogout`
   - **问题**: 登出后token未被正确撤销
   - **影响**: 安全风险，已登出的token仍可使用
   - **修复建议**: 验证`security.revoke_token`实现
